# 新手指南（从零开始，一步一步来）

这份给完全没用过这类工具的人。你只要会复制粘贴、会打字，就能跟着走完。看不懂的词不用管，照做就行。

先确认两件事：你有一台电脑（Windows 或 Mac 都行），能上网。够了。

## 先选一条路

电子军师有两种用法，挑一个：

- **路线一 · 装进 Claude Code（下面「跟着装好 Claude Code」）**：跟 AI 对话着用，功能最全，能长期记住每个对象。第一次装 AI 工具的推荐这条。
- **路线二 · 桌面 App（下面「用桌面 App」）**：一个像微信的聊天窗口，左边对象列表、中间三条能发的回复、右边军师面板。想接 DeepSeek 或 GLM（智谱）而不是 Claude 的走这条。

两条路各看各的，不用都做。

---

## 跟着装好 Claude Code

要在电脑上粘几行命令，不难，每行都给你写好了。（在用 Codex 的不用看这份，照 [README](../README.md) 里的「路线 B · Codex」一行命令就行。）

### 第 1 步：打开「终端」

终端就是一个黑色的小窗口，命令粘进去回车就行。

- **Windows**：按键盘左下角的 `Win` 键，同时按 `X`，弹出一个菜单，点里面的「终端」。如果没有「终端」，点「Windows PowerShell」也一样。
- **Mac**：按 `Command + 空格`，屏幕中间出来一个搜索框，打字「终端」（或者「Terminal」），回车，黑窗口就开了。

### 第 2 步：装 Claude Code

把下面这行整段复制，粘到黑窗口里，回车。等它跑完（可能要一两分钟）。

- **Windows**：

  ```powershell
  irm https://claude.ai/install.ps1 | iex
  ```

- **Mac**：

  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```

跑完之后，它可能让你登录。按提示用 Claude 账号登一下；没有账号就照着提示免费注册一个。

### 第 3 步：装电子军师

还在同一个黑窗口里，再复制下面这行，粘进去回车：

- **Windows**：

  ```powershell
  irm https://raw.githubusercontent.com/shoal-rat/dianzi-junshi/master/install.ps1 | iex
  ```

- **Mac**：

  ```bash
  curl -fsSL https://raw.githubusercontent.com/shoal-rat/dianzi-junshi/master/install.sh | bash
  ```

看到「装好了」就成了。

### 第 4 步：进去开聊

在黑窗口里打 `claude`，回车。等它进入对话界面后，直接打字：

```text
帮我追个人
```

它会问你几个小问题：给对象起个代号、你俩现在到哪一步、要不要开「反舔狗模式」（明显没戏时直接拉住你）。不知道就回「跳过」。

建好之后，把你和 ta 的微信截图丢进去，或者把聊天记录复制粘贴进去，它就给你回复。

### 平时怎么用

进去（打 `claude` 回车）之后，正常说人话就行：

- 「这条我该怎么回」（配一张截图）
- 「ta 这么说什么意思，我还有戏吗」
- 「我想发『在吗』，行不行」
- 「ta 答应周末见面了，帮我安排」
- 「换个人」或「新建」：同时追几个，互不影响

## 出问题了？

**粘命令没反应 / 报红字**
先确认你是整段复制的，没漏字。Windows 如果提示找不到 `git`，去 [git-scm.com](https://git-scm.com/downloads/win) 下载安装（一路点下一步），装完重开黑窗口，再做一遍第 3 步。

**打 `claude` 说找不到这个命令**
把黑窗口关掉、重新打开一次再试。第 2 步装完有时要重开窗口才生效。

**进去了，但它不认识「电子军师」/ 不理我**
把 Claude Code 整个退出（在对话里打 `/exit` 回车，或直接关窗口），重新打 `claude` 进去。新装的技能要重启一次才认。进去还是不认，就直接打 `/dianzi-junshi` 回车，把它叫出来。

**它问的问题我答不上来**
全都能回「跳过」。它会先按常规来，等你发了截图再慢慢摸清楚。

**截图怎么发给它**
把图片直接拖进对话窗口，或者复制粘贴进去就行，Claude Code 能直接看图，不用你描述。

## 用桌面 App

想要一个像微信的窗口、还想接 DeepSeek 或 GLM（智谱）的走这条。要先装一个叫 Bun 的小工具（跑 App 用），再启动 App。

### 第 1 步：装 Bun

打开「终端」（怎么开见上面第 1 步），粘这行回车，等它跑完：

- **Mac / Linux**：

  ```bash
  curl -fsSL https://bun.sh/install | bash
  ```

- **Windows**（PowerShell）：

  ```powershell
  powershell -c "irm bun.sh/install.ps1 | iex"
  ```

装完把终端关掉重开一次。

### 第 2 步：下载电子军师

还在终端里，粘这行回车（把整个项目下载到你的电脑）：

```bash
git clone https://github.com/shoal-rat/dianzi-junshi.git
```

提示没有 `git` 的，先去 [git-scm.com](https://git-scm.com/downloads) 装一下再回来。

### 第 3 步：启动 App

```bash
cd dianzi-junshi/app
bun start
```

浏览器会自己打开一个页面（打不开就手动访问 `http://localhost:5177`）。

### 第 4 步：接上模型

第一次进去是「演示模式」——回复是内置的示例，先让你看看长啥样。想让它真的读你的消息：

1. 点右下角那颗「演示模式」的小胶囊。
2. 选 Claude、DeepSeek 或 GLM（智谱）里你有账号的一个。
3. 把 API key 填进去（每个选项旁边有「去获取」的链接，点进去在对应官网申请）。DeepSeek 和 GLM 通常比 Claude 便宜，国内也好付款。
4. 保存。之后新建对象、贴截图、出方案，跟微信一样用。

> 想把它变成一个双击就能开的应用（不用每次敲命令），在 `app` 目录里跑 `bun run build:app`，会在 `app/dist/` 生成一个单独的可执行文件，拷到哪台电脑都能直接跑。

App 的更多细节看 [app/README.md](../app/README.md)。
