# 电子军师桌面版新手指南

不需要会编程，也不用学提示词。整个过程就是：安装、建档、贴聊天。

## 1. 下载

打开 [最新 Release](https://github.com/shoal-rat/dianzi-junshi/releases/latest)。

- Windows：下载名字以 `setup.exe` 结尾的文件。
- 新款 Mac（M1/M2/M3/M4 等）：下载名字含 `aarch64` 的 DMG。
- 旧款 Intel Mac：下载名字含 `x64` 的 DMG。
- Ubuntu / Debian：下载 DEB。
- 其他 Linux：下载 AppImage。

双击安装。系统第一次出现安全提醒时，先确认文件来自本项目的 GitHub Release。

## 2. 第一次建档

打开电子军师后，只需要先写一个称呼，例如“小林”或“同事 A”。称呼只是为了不把不同人的聊天弄混，不会被当作关系事实。

关系进度拿不准时，选「有点熟 / 还没确定」即可。以后可以随时修改。

## 3. 带入以前的资料

你可以：

- 粘贴以前的聊天文字；
- 把很多张微信聊天截图一起拖进去；
- 放朋友圈、小红书或其他平台的截图；
- 写下重要约定、偏好和见面经历。

截图会一张张保存和整理。进度可以收起，任务不会因此取消。没有选好 AI 时，图片先留在本机等待。

## 4. 选择 AI

最省事的选择：

- 电脑已经登录 Codex，就选择 Codex；
- 已经登录 Claude Code，就选择 Claude Code；
- 两者都没有，可以配置自己使用的 API；
- 只想看看界面，可以先用演示模式。

## 5. 正常说话就行

```text
这句怎么回
只帮我读懂 ta，不要生成回复
我想这样说，帮我改自然一点
看看最近有没有真的在推进
```

每条建议旁边都有复制按钮。发出去以后，可以点「记录后来结果」，告诉 App 对方是接住了、普通回复、变冷，还是一直没回。

## 6. 为什么要记结果

猜测不能让判断变准，真实后续可以。电子军师会观察：

- 对方最近回应是不是变快或变慢；
- 会不会主动推进；
- 答应的事情有没有做到；
- 是否记得以前说过的细节；
- 哪一类表达更容易得到好结果。

一次结果不会给人定性。旧证据会随时间变弱，近期变化和长期习惯会分开显示。

## 7. 关掉以后资料还在吗

在。资料保存在这台电脑的 `~/.dianzi-junshi/`。卸载 App 前如要彻底删除资料，需要另外删除这个目录。
