# 新手指南（从零开始，一步一步来）

这份给完全没用过这类工具的人。你只要会复制粘贴、会打字，就能跟着走完。看不懂的词不用管，照做就行。

先确认两件事：你有一台电脑（Windows 或 Mac 都行），能上网。够了。

---

## 先选一条路

- **想要最好用、能长期记住每个人** → 看下面的「路线 A：Claude Code」。要在电脑上粘几行命令，不难，每行都给你写好了。
- **一行命令都不想碰** → 跳到最后的「路线 B：只用 ChatGPT」。

---

## 路线 A：Claude Code

### 第 1 步：打开「终端」

终端就是一个黑色的小窗口，命令粘进去回车就行。

- **Windows**：按键盘左下角的 `Win` 键，同时按 `X`，弹出一个菜单，点里面的「终端」。如果没有「终端」，点「Windows PowerShell」也一样。
- **Mac**：按 `Command + 空格`，屏幕中间出来一个搜索框，打字「终端」（或者「Terminal」），回车，黑窗口就开了。

### 第 2 步：装 Claude Code

把下面这行整段复制，粘到黑窗口里，回车。等它跑完（可能要一两分钟）。

- **Windows**：

  ```powershell
  irm https://claude.ai/install.ps1 | iex
  ```

- **Mac**：

  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```

跑完之后，它可能让你登录。按提示用 Claude 账号登一下；没有账号就照着提示免费注册一个。

### 第 3 步：装电子军师

还在同一个黑窗口里，再复制下面这行，粘进去回车：

- **Windows**：

  ```powershell
  irm https://raw.githubusercontent.com/shoal-rat/dianzi-junshi/master/install.ps1 | iex
  ```

- **Mac**：

  ```bash
  curl -fsSL https://raw.githubusercontent.com/shoal-rat/dianzi-junshi/master/install.sh | bash
  ```

看到「装好了」就成了。

### 第 4 步：进去开聊

在黑窗口里打 `claude`，回车。等它进入对话界面后，直接打字：

```text
帮我追个人
```

它会问你几个小问题：给对象起个代号、你俩现在到哪一步、要不要开「反舔狗模式」（明显没戏时直接拉住你）。不知道就回「跳过」。

建好之后，把你和 ta 的微信截图丢进去，或者把聊天记录复制粘贴进去，它就给你回复。

### 平时怎么用

进去（打 `claude` 回车）之后，正常说人话就行：

- 「这条我该怎么回」（配一张截图）
- 「ta 这么说什么意思，我还有戏吗」
- 「我想发『在吗』，行不行」
- 「ta 答应周末见面了，帮我安排」
- 「换个人」或「新建」：同时追几个，互不影响

## 出问题了？

**粘命令没反应 / 报红字**
先确认你是整段复制的，没漏字。Windows 如果提示找不到 `git`，去 [git-scm.com](https://git-scm.com/downloads/win) 下载安装（一路点下一步），装完重开黑窗口，再做一遍第 3 步。

**打 `claude` 说找不到这个命令**
把黑窗口关掉、重新打开一次再试。第 2 步装完有时要重开窗口才生效。

**进去了，但它不认识「电子军师」/ 不理我**
把 Claude Code 整个退出（在对话里打 `/exit` 回车，或直接关窗口），重新打 `claude` 进去。新装的技能要重启一次才认。

**它问的问题我答不上来**
全都能回「跳过」。它会先按常规来，等你发了截图再慢慢摸清楚。

**截图它看不了**
确认你用的是能看图片的环境。把图片直接拖进对话，或者复制粘贴进去。

## 路线 B：只用 ChatGPT（不装任何东西）

已经在用 ChatGPT 网页或 App 就走这条，一行命令都不用。

1. 打开 [platforms/chatgpt-instructions.md](../platforms/chatgpt-instructions.md)。
2. 照着把那段「Instructions」复制，粘进 ChatGPT 的自定义指令，或者新建一个自定义 GPT 贴进去。
3. 然后就能聊了：发截图、贴聊天记录，让它帮你分析、给回复。

缺点：ChatGPT 默认读不到你电脑里的文件，换个对话也容易忘事。想要长期记住每个对象、自动整理资料，还是路线 A 更顺。
