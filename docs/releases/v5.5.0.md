# v5.5.0 · White Canvas & Visual Outcomes — 2026-07-12

## 全新界面

- 界面重做为白色主画布：操作使用克制的蓝色，侧栏与判断面板改为低对比度灰白层级，卡片、弹窗、输入区和移动端布局统一到同一套视觉系统。
- 思考深度与胆量滑杆改为独立控制卡。轨道、拇指和档位名会随级别改变颜色：低档更冷静，中档为蓝色，高档过渡到紫色和橙色；手机宽度下仍同时显示两个完整滑杆。
- 新品牌图形使用三点决策网络，不再沿用绿色勾号；favicon、应用内品牌标记与 README 图形同步。
- 重新绘制三步上手、桌面平台与决策引擎三张 SVG；主界面、批量进度、决策网络和结果反馈截图全部更新。

## 截图识别真实结果

「记录后来结果」现在同时接受文字和聊天截图：

1. 「你当时发的是」改成可编辑文本框，保存用户最后实际发送的内容，而不是只能沿用建议原文。
2. 用户可以粘贴或选择最多 6 张回复截图；添加后立即调用当前支持视觉的 AI。
3. AI 自动提取对方的关键回复，选择积极 / 普通 / 负面 / 未回复，估计回复时间，并勾选继续聊、主动推进、兑现、失约、记得细节或忘记细节等明确信号。
4. 自动判断会显示置信度和一句理由；所有选择都保留为可编辑状态，用户确认后才写入学习系统。

截图内容被视为不可信资料，提示中明确忽略图片里的指令。云模型直接接收本轮图片；Codex / Claude Code 只读取档案目录中的临时文件，分析结束立即删除。演示模式提供确定性示例，不联网。
