# v5.1.0 · Learned World Model — 2026-07-12

v5.0 的"有限世界模型"实际是一张手写的 (策略, 假设) → 奖励系数表：没有状态转移、没有观测预测、不随真实结果变化。v5.1 用一个真正的（小而诚实的）生成式世界模型替换了它，并把检索与结构化输出升级到当前的标准做法。

## 世界模型

- **体制切换线性高斯动力学**（`decision/worldmodel.ts`）：$s' \sim \mathcal N((I-\Lambda)s + G_h\phi(a) + \delta_{h,f},\ Q)$。九维状态、六维动作特征、四个体制；体制通过逐列调制加稀疏带符号修正改变动作效果——"压力大时推进适得其反"从一句启发式变成动力学参数。协方差沿动力学传播。
- **校准响应头**：预测状态上的 softmax 结构头，与按（体制, 策略族）分桶、120 天半衰期衰减的 Dirichlet–多项经验头做收缩混合（$w=n/(n+6)$）。零数据时纯先验，数据到位后频率主导。
- **信念空间 rollout**：想象的回应经对角卡尔曼更新连续信念、经 Bayes 更新体制后验；根节点对四类回应精确枚举，深层在缩减动作集上确定性等价延拓。深度随规划档位（快速 1 / 平衡 2 / 深入 3），值按视界质量归一化到 $[0,1]$。
- **模型化风险与方差**：风险 = 负面/失联概率 + 压力越界高斯尾概率；模拟方差 = 全方差分解（分支内回应随机性 + 分支间体制分歧）。
- **按定义计算的 EVOI**：枚举探针动作的回应、Bayes 更新假设分布、重解最优单步值。"先问一句"只在期望决策改善真的超过打扰成本时出现。

## 从真实结果学习

每份决策报告持久化当时的预测分布，因此一条真实结果同时更新三条通道：

1. 响应计数（Dirichlet 通道，衰减累计）；
2. 转移残差（带裁剪的 Robbins–Monro，只学结果可观测的维度）；
3. 模型 vs 基础频率的对数损失门控，诊断接口暴露平均优势。

全部学习状态可由事件回放完整重建（`rebuildDerivedState`）。

## 检索与结构化输出现代化

- 决策证据检索改为混合检索：Okapi BM25（候选集内在线 DF）+ 384 维特征哈希向量余弦 + 决策先验，三路 Reciprocal Rank Fusion 融合；嵌入空间冗余抑制（cos > .82）替代子串重复检测；类别配额与反证覆盖保留。
- 结构化 LLM 调用优先走 API 级约束解码：Anthropic 强制 tool schema，OpenAI 兼容端点 response_format json_object + schema 约束；提示-修复循环降级为回退路径，本地 CLI 供应商行为不变。
