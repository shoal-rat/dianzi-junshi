# v5.2.0 · Decision Network & Boldness — 2026-07-12

v5.1 给了引擎一个真正的世界模型；v5.2 把它亮出来给人看，并把风格控制权交给用户。

## 决策网络可视化

每份决策报告新增 `networkTrace`：引擎本轮真实执行的分层数据流。

- 层：状态信念 → 体制假设 → 选中策略 → 动作特征 φ(a) → 预测状态 s′ → 回应分布 p(o|s′,h,f) → 期望价值 / 风险 / 不确定性。
- 每个节点的激活值、参数明细（均值±方差、有效样本、后验概率、学习计数等）与流入/流出的贡献权重全部来自当轮计算，点开即可对账。
- 边权重来自真实系数：信念→体制用假设打分权重，特征→状态用体制混合后的增益 G_h·φ，状态→回应用响应头载荷 u_o，回应→输出用效用与风险权重。
- 前端零依赖 SVG 渲染，随浅色/深色主题自动配色；生成回复时即可点开观看（决策先于表达完成）。

## 胆量与思考深度滑杆

顶栏两个滑杆直接控制引擎，设置按聊天档案保存：

- **思考深度**：快速 / 平衡 / 深入，对应 rollout 视界 1/2/3 与体制分支 2/3/4。
- **胆量 β∈[0,1]**：重塑规划目标，模型预测保持不变——
  - 风险惩罚 ×(1.45−0.9β)，信息价值权重 ×(1.2−0.4β)
  - 追问成本 ×(0.6+0.8β)：越大胆越倾向直接行动而不是先问
  - 弃权门槛 0.76+0.28(β−½)：大胆档容忍更高不确定性
  - invite/direct 获得正向情境加成，give_space 反向；探索奖励 ×(0.7+0.6β)
  - 表达层语气随档位调整（更敢给钩子 / 更留余地）
