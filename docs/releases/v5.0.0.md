# v5.0.0 · Decision Desktop

这次更新把电子军师从“把上下文交给模型写答案”升级成可检查、可学习、可回放的本地决策应用。

## 对普通用户有什么变化

- 回复前会先比较几种可能解释和多种策略。
- 信息不够时会直接说不确定，并问最关键的一点。
- 右侧可以展开查看状态、候选方案、证据和风险；默认界面仍然简单。
- 可以选快速、平衡或深入思考。
- “记录后来结果”现在会准确关联本次决策、策略和文案。
- 所有弹窗都支持关闭按钮、背景点击和 Esc。

## 引擎更新

- SQLite 事件源、模式迁移和投影回放
- 九维时间信念、短期/长期衰减、变化和冲突检测
- 竞争性状态假设与缺失信息识别
- 决策用途检索、时间事实和冲突保留
- 策略 taxonomy、有限世界模型、多批评器和可配置奖励
- VOI 追问、三档预算、不确定性与保守选择
- 带衰减的上下文 Beta bandit 和有界探索
- 结构化输出校验、修复、重试、缓存和确定性回退
- 决策诊断、历史查询和真实事件回放
- 离线合成评测与 CI 门槛
- 明确同意的去标识化真实结果校准、导出和撤回
- 时间证据图、关系有效期与可控模式生命周期
- 历史策略表现和 30 天趋势面板
- macOS Keychain、Windows Credential Manager、Linux Secret Service
- 条件式 Apple notarization、Windows Authenticode 和构建来源证明

## 兼容性

v4 的聊天、截图记忆和反馈会原样保留。旧反馈在首次启动时自动投影到新事件层。数据库升级是追加式的，不删除旧表。

## 验证

```bash
cd app
bun install --frozen-lockfile
bun run verify
```

桌面安装包继续由 GitHub Actions 为 macOS、Windows 和 Linux 分平台构建。签名与公证需要仓库维护者提供平台证书。
