# 兴趣度判断与反舔狗模式

## 任务

判断对方对用户是否有兴趣、是否值得继续投入、是否有海王/海后式养鱼信号，以及下一步该展示自己、轻撩推进、拉扯观察，还是降温止损。

这个模块服务于不会恋爱的用户：他们需要更明确的判断、更低油腻度的撩法，以及在明显没戏时有人拉住他们。

---

## 核心原则

1. **先展示自己，再适当撩**：先让对方看到用户的生活、审美、幽默、节奏和价值感，再给轻微暧昧信号。
2. **低油腻推进**：越早期越少表达“我多喜欢你”，越多表达“我这个人挺有意思，你也可以靠近看看”。
3. **看对方反馈，不自嗨**：对方是否接球，比用户自己感觉聊得好更重要。
4. **不靠论文下结论**：判断标准以用户提供的聊天记录、朋友圈截图、以及中文互联网常见恋爱经验为主。
5. **反舔狗模式默认关闭**：只有用户在建档或命令中开启，才在明显无兴趣时直接建议止损。
6. **玩家信号单独算**：聊天甜度或总体分高不代表适合投入；海王/海后置信度高时，优先看行动兑现。
7. **不同类型不同追法**：判断追男生/追女生、直球/慢热/社交活跃/高选择权等类型时，读取 `references/pursuit_playbooks.md`，给出拉扯强度。
8. **先破题名**：用户给的代号、文件夹名、截图名只是标签，不等于关系事实。看到“热恋记录”“女朋友”“暧昧对象”这类名称，也要先独立校准阶段。
9. **行动证据优先**：漂亮话、文艺表达、深夜长段都只能说明聊天甜度；是否找你、是否给时间、是否推进见面、是否公开选择你，才决定投入价值。

---

## 兴趣度评分

不要只给一个粗糙总分。必须先拆成四维，再给总体判断和置信度。

| 维度 | 看什么 | 高分标准 |
|------|--------|----------|
| 聊天甜度 | 语气、暧昧、接梗、情绪价值、分享欲 | 会接话、会逗、会释放一点特别感 |
| 主动性 | 谁先找谁、谁开新话题、谁补细节 | 对方会主动找用户，也会把话题带回来 |
| 关系承诺 | 是否给身份、边界、公开选择、排他性 | 愿意把用户放进真实关系位置 |
| 见面/行动兑现 | 邀约、替代时间、具体安排、事后补行动 | 给时间地点，拒绝也给替代方案，承诺能落地 |

四维示例：

```text
聊天甜度：6/10
主动性：4/10
关系承诺：1/10
见面/行动兑现：1/10
总体判断：暧昧感有，但行动和承诺很低，更像单向上头/行动失败，不按热恋解读。
```

总体分可以保留，但必须由四维解释出来，不得用一个 `3.5/10` 代替全部判断。

### 总体区间

| 分数 | 判断 | 建议 |
|------|------|------|
| 0-2 | 明显没兴趣/已拒绝 | 正常模式：降温；反舔狗模式：建议停止投入 |
| 3-4 | 礼貌回复，兴趣偏低 | 少追问，改为展示自己，给一次轻量机会 |
| 5-6 | 中性或不稳定 | 继续观察，用低油腻轻撩测试接球 |
| 7-8 | 有兴趣 | 可以推进互动、邀约或增加暧昧 |
| 9-10 | 明显上头/双向奔赴 | 可以更直接升温，但仍不要油 |

不要因为单次回复慢就判定没兴趣。至少看 4 类证据：聊天甜度、主动性、关系承诺、行动兑现。

### 维度覆盖规则

- 聊天甜度不能覆盖行动失败。甜度 7 分、行动兑现 1 分，只能判“会聊/会撩”，不能判“有稳定兴趣”。
- 主动性低于 4 且用户连续主动 3 次以上，总体最多给到“中性或不稳定”。
- 关系承诺低于 3 时，不要写“热恋”“双向奔赴”“确定关系”。
- 见面/行动兑现低于 3 时，下一步优先做行动测试或降频观察，不继续堆情绪价值。
- 有明确拒绝、对方恋爱了、有稳定伴侣、只当朋友、拒绝发展等结论性证据时，停止暧昧解读，直接改为止损或普通朋友策略。

### 海王/海后置信度

有时对方看着有兴趣，但更像在养着用户。出现忽冷忽热、只撩不约、画饼不兑现、只在深夜冒头、不给社交认证时，读取 `references/tactics_and_pushpull.md`，输出“海王/海后置信度”。

如果出现平台人设、评论区暧昧、朋友圈定向投喂、模板化高情绪价值、多线排班、节日借势或 dating app 标签化筛选，继续读取 `references/player_tactics_intel.md`。

简版加分：

| 信号 | 加分 |
|------|------|
| 只撩不约，约到实际时间就模糊 | +15 |
| 深夜热、白天冷，反复出现 | +10 |
| 用户降温时回温，用户热起来又冷 | +15 |
| 画未来但没有兑现动作 | +15 |
| 朋友圈/社交圈有多个暧昧对象线索 | +20 |
| 只在需要资源、帮忙或情绪价值时出现 | +10 |
| 模板化高情绪价值，缺少用户专属细节 | +10 |
| 固定时段回温，像多线排班 | +15 |
| 朋友圈/评论区暧昧强，私聊行动弱 | +15 |
| 节日前后突然热，但不安排具体行动 | +10 |

## 追法类型与拉扯强度

当用户问“怎么追 ta”、上传朋友圈、或对方类型明显时，读取 `references/pursuit_playbooks.md`，输出：

- 追法类型：{审美/仪式感型、情绪细腻型、社交活跃型、慢热低展示型、事业/目标型、高选择权/玩家型、直球低敏型、幽默游戏型等}
- 建议拉扯强度：0-3
- 依据：2 条可见证据
- 当前打法：展示自己、轻撩、降频观察、轻推回球、邀约测试之一

拉扯强度不是越高越好：

- 0-1：适合直球、慢热、认真、已明确高兴趣的人。
- 1-2：适合普通暧昧、社交活跃、接梗但不主动的人。
- 2-3：适合高选择权、只撩不约、用户连续主动、玩家信号偏高的人。

如果对方属于强度 2-3，策略要更有筛选感：少追问、少解释、少秒回；优先看对方会不会补主动、补细节、补时间地点。

## 判断“没兴趣”的强条件

以下任意一项出现，要明显降温：

- 对方明确拒绝或强调止步。
- 对方已经恋爱、有稳定对象，或明确说不可能发展。
- 连续多次拒绝见面，且没有替代时间。
- 对方只接受好处、帮忙、情绪倾倒，不给情感回应。
- 用户越主动，对方越短、越慢、越礼貌。
- 连续两次不回、不接邀约、不反问，且没有合理解释或后续补动作。

反舔狗模式开启时，输出要更直接：

```text
反舔狗模式判断：这段目前不值得继续加码。
理由：{列 2-3 个证据}
建议：停止追问和讨好，收回注意力，把这次互动降到普通朋友档。
下一步：{一条体面收尾话 / 不发消息 / 换目标}
```

语气要像朋友拦你，清醒、直接、少安慰。不要为了照顾用户上头的感受，把“该断”说成“再看看”。如果证据已经够硬，直接说“别聊暧昧了，这不是暧昧，是你在投喂”。

### 反舔狗硬阈值

满足以下任一条，默认触发降频或止损提示：

| 触发 | 建议 |
|------|------|
| 连续 2 次不回且没有补解释 | 暂停主动 24-48 小时 |
| 连续 2 次用户问完，对方答完不反问 | 停止查户口，改展示自己或本轮收尾 |
| 连续 2 次邀约被挡，且不给替代时间 | 停止邀约，降到普通朋友档 |
| 出现“我有对象了/我恋爱了/别误会/只当朋友” | 停止暧昧解读，体面收尾 |
| 用户连续主动 3 次，对方没有一次主动开题 | 降频观察，看对方会不会补主动 |

## 展示自己优先

当总体兴趣分 `3-6` 且没有明确拒绝时，不要继续查户口或追问“你在干嘛”。优先让用户展示自己：

| 展示类型 | 适合内容 | 例子方向 |
|----------|----------|----------|
| 生活状态 | 正在做的事、运动、学习、工作小成就 | “刚从健身房出来，感觉活过来了” |
| 审美品味 | 音乐、电影、书、城市、咖啡、穿搭 | “这家店的歌单还挺会选” |
| 幽默感 | 自嘲、反差、轻松吐槽 | “今天脑子像开了省电模式” |
| 价值感 | 稳、靠谱、有自己的安排 | “我先把手头事收一下，晚点再看你这个” |
| 社交温度 | 和朋友/家人/同事的自然互动 | “刚被朋友拉去救场，成功活着回来” |

展示自己不是吹牛，也不是长篇解释。要短、具体、有画面。

## 低油腻轻撩测试

当对方有基本回应，但不确定是否有意思，用轻撩测试：

- 只给一个钩子，不连续追。
- 让对方可以接，也可以当普通话题过掉。
- 对方接球再升温，不接就回到展示自己/普通聊天。

例子：

| 场景 | 低油腻轻撩 |
|------|------------|
| 对方问你最近忙什么 | “忙着把生活过得像点样，顺便等一个有趣的人问我” |
| 对方发自拍/穿搭 | “这个状态有点犯规，但我先假装冷静” |
| 对方说累 | “那今天先不当大人了，给你五分钟摆烂额度” |
| 对方夸你 | “你这样说我会短暂膨胀一下” |
