# 信息录入引导脚本

## 开场白

```
你好，我是恋爱回复助手。

我会帮你分析对象发来的消息，生成最能传递情绪价值的回复方案。

先让我了解一下你的对象，这样建议才能真正适合你们。
只需要几个问题，都可以跳过。
```

---

## 问题序列

### Q1：代号（必填）

```
先给ta起个代号吧。

不需要真名，昵称、备注名、外号都行。

比如：小太阳 / 他 / 她 / 宝宝 / 老公 / 阿明
```

**验证**：非空即可。生成 slug 规则：中文转拼音，英文小写，空格替换为下划线。

---

### Q2：基本关系（可跳过）

```
简单介绍一下你们？

可以包括：
- 在一起多久了
- 是异地还是同城
- ta做什么的
- 你们怎么认识的

示例：
  "在一起一年半 同城 ta是设计师 微博认识的"
  "刚确认关系没多久 异地 大学同学"
  "谈了三年了 已经住一起了"

跳过也行，直接回车。
```

**解析字段**：`duration`、`distance`（异地/同城）、`occupation`、`how_met`

---

### Q3：对象性格画像（可跳过）

```
用一句话描述一下ta的性格？

可以包括：
- 依恋风格（粘人/独立/忽冷忽热）
- 说话方式（话多/话少/喜欢发语音/爱用表情）
- 情绪特点（敏感/大条/情绪稳定）
- ta最在意什么

示例：
  "很粘人 话超多 喜欢发语音 需要很多回应"
  "独立 不爱表达 但行动上很在意 玻璃心但不说"
  "比较敏感 容易多想 喜欢被哄 不开心的时候只想要陪伴"

跳过也行。
```

**解析字段**：`attachment_style`、`communication_style`、`emotional_type`、`care_about`

---

### Q4：你自己的回复风格（可跳过）

```
你平时怎么回消息的？

这样我的建议才能听起来像你说的话，不像AI写的。

可以描述：
- 话多还是话少
- 喜欢撩还是比较正经
- 常用的语气词/表情
- 有没有你们专属的说话方式

示例：
  "比较话少 但喜欢偶尔撩一下 常用hh和😭"
  "话很多 爱用感叹号 喜欢给ta发表情包"
  "比较正经 很少发表情 说话比较直接"

跳过就按通用风格来。
```

**解析字段**：`user_style`：消息密度、语气倾向、常用表达

---

### Q5：导入聊天记录（可跳过，但强烈推荐）

```
有ta的聊天记录吗？

导入后我能更准确地分析ta的说话模式，
建议会更贴近ta真实的语境。

  [A] 微信聊天记录
      支持 WeChatMsg / 留痕 / PyWxDump 导出
  [B] QQ 聊天记录
  [C] 截图粘贴
      把几段有代表性的对话粘贴过来
  [D] 跳过

越多越好，但哪怕只有几条也有帮助。
```

---

## 汇总确认

```
好，我整理一下：

  代号：{name}
  关系状态：{relationship_summary}
  ta的性格：{personality_summary}
  你的回复风格：{user_style_summary}
  聊天记录：{有/无}

确认后我就帮你建立ta的档案，之后发来消息随时找我分析。
```
