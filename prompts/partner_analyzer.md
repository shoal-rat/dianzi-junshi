# 对象分析器

## 任务

从聊天记录、截图和用户描述中提取对象的沟通模式和情感需求，构建可驱动回复建议的 Partner Profile。

---

## 提取维度

### 1. 说话风格

- **消息密度**：话多还是话少？会不会连发好几条？
- **消息长度**：短句为主？长段落？还是一句话带所有信息？
- **语气词习惯**：常用哪些？（嗯/哦/哈哈/啊/呀/唉...）
- **标点风格**：用不用句号？感叹号多不多？省略号？
- **emoji/表情包**：爱用什么表情？有没有标志性的？
- **语音习惯**：喜欢发语音还是打字？
- **称呼方式**：怎么叫你？有没有专属称呼？

### 2. 情感需求模式

- **爱的语言**：最能让ta感受到爱是哪种？
  - 肯定的言辞（听到你喜欢我、你最好了）
  - 精心的时刻（陪伴、专注、在场）
  - 接受礼物（惊喜、礼物、小心意）
  - 服务的行动（帮ta做事、记住ta提过的事）
  - 身体接触（抱抱、牵手、亲密）
- **被回应的需求**：
  - ta希望你多快回？
  - ta在意你回复的长度吗？
  - 如果你不回或回得敷衍，ta会怎么反应？
- **被看见的需求**：
  - ta分享事情时，最在意你怎么回应？（夸ta/给建议/一起骂/只是听着）
  - ta表达负面情绪时，ta需要的是安慰还是解决方案？

### 3. 高频行为模式

- **撒娇方式**：怎么撒娇？会不会直接说？
- **生气方式**：生气时是冷处理还是直接说？会不会发很多"？"或"？？？"？
- **不开心的信号**：ta不开心时的典型表现是什么？（话变少/语气变冷/发奇怪的表情）
- **开心时的样子**：ta高兴时怎么表现？（话变多/连发消息/分享欲变强）
- **试探行为**：ta有没有说话"拐弯抹角"、说了又不说的习惯？

### 4. 关系动态

- **谁更主动**：在关系中谁更主动联系？
- **不安全感来源**：ta什么时候会多想？（你回复慢/你不主动/你说话很短）
- **被满足的时刻**：ta什么时候表现得最开心？（你夸ta/你记住了某件事/你主动关心）
- **地雷话题**：什么话题/说法容易让ta不舒服？

### 5. 潜台词模式（重要）

很多消息的表面含义和实际需求是两层：

| 表面说的 | 可能真正需要 |
|---------|------------|
| "没事" "我挺好的" | 需要你主动追问 |
| "随便" "都行" | 有想法但想让你猜/主动 |
| "你忙吧" "不打扰你了" | 其实很想你陪 |
| "算了" | 可能很委屈但不知道怎么说 |
| "你好烦啊" "去死啦" | 在撒娇或在开玩笑 |
| "我最近很累" | 需要被关心，不只是说说 |
| "你怎么不主动" | 需要你展示主动和在意 |

记录对象的个人化潜台词模式。

---

## 输出格式

```markdown
## 对象档案：{name}

### 沟通风格
- 消息密度：{高/中/低}，{具体描述}
- 语气特征：{口头禅列表}
- 常用表情：{列表}
- 语音习惯：{有/无/偶尔}

### 情感需求
- 主要爱的语言：{类型 + 具体表现}
- 被回应需求：{描述}
- 被看见需求：{描述}

### 高频行为模式
- 撒娇：{描述}
- 生气/不开心信号：{描述}
- 开心信号：{描述}
- 试探模式：{描述或"无明显试探"}

### 关系动态
- 主动方：{ta/你/大致持平}
- 多想触发器：{列表}
- 满足感来源：{列表}
- 地雷话题：{列表}

### 潜台词字典
（ta专属的说话习惯解码）
| ta说的 | 可能需要的 |
|--------|---------|
| {具体说法} | {解读} |
```
