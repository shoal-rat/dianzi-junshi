# 朋友圈/社交动态画像分析

## 任务

根据用户上传的朋友圈截图、社交动态截图、照片说明、个签、聊天记录和用户描述，分析对方的生活方式、性格展示、社交类型、情绪表达方式和适合的聊天策略。

注意：朋友圈是“对外展示面”，不是完整人格。只能输出“可能的展示倾向”，不能下诊断式结论。

---

## 支持输入

- 朋友圈/小红书/微博/Instagram 等截图。
- 用户复制的动态文案。
- 图片内容描述。
- 既往聊天记录或 `tools/wechat_parser.py` 生成的报告。

必须直接看图片，不要只做 OCR。朋友圈大部分信息在图片里，不在文字里：妆容、穿搭、滤镜、场景、同框的人、评论区，都要看进去。看原图全尺寸，不要用缩略图——妆容、配饰、评论里的小字这些缩略图会糊掉；字密或图大就分区域放大看（见 SKILL.md「图片读取规则」）。

如果一次收到多张图，不要只抽样看“关键图”。每张图都要形成一条结构化观察记录，并写入或更新 `partners/{slug}/materials/image_observations.jsonl`。最终结论必须能追溯到具体图片。

---

## 分析维度

### 0. 单图留痕

对每张图片先做一条记录，再综合判断：

- `image_id`：沿用导入模板里的编号；没有模板就临时编号。
- `visible_context`：朋友圈、聊天截图、自拍、合照、评论区、未知。
- `initiator`：如果是聊天截图，记录谁主动开启本轮；不是聊天图写 `not_applicable`。
- `receiver_response`：对方有没有接球、转移、错过、只礼貌回应。
- `has_counter_question`：有没有反问或把话题带回用户。
- `has_invite` / `invite_detail`：有没有邀约、时间地点是否具体。
- `action_fulfillment`：已兑现、待兑现、失败、未知、不适用。
- `warming_signals`：主动补细节、接梗、分享生活、给时间等。
- `cooling_signals`：只答不问、拒约不给替代时间、长时间不回、转移话题等。
- `visible_evidence`：只写图上看得到的东西，不写脑补。

综合结论必须引用这些记录，例如“img_0004 有接梗但无反问，img_0011 拒约不给替代时间，所以聊天甜度高于行动兑现”。

### 1. 展示主题

| 类型 | 常见内容 | 可能聊天切入 |
|------|----------|--------------|
| 生活记录型 | 吃饭、通勤、日常小事 | 轻松共鸣，接日常细节 |
| 审美品味型 | 咖啡、展览、音乐、穿搭、旅行 | 夸具体品味，不空泛夸人 |
| 成就展示型 | 工作、学习、证书、健身成果 | 真诚认可 + 轻松不跪舔 |
| 情绪表达型 | 文案、深夜、emo、歌词 | 先接情绪，不急着撩 |
| 社交活跃型 | 聚会、朋友、活动 | 展示自己的生活，不要查岗 |
| 低展示/神秘型 | 很少发、内容克制 | 低油腻，少问隐私，多看聊天反馈 |

### 2. 性格/类型假设

只能输出“可能偏向”，并给证据：

- 外放分享型：动态多、细节多、情绪表达直接。
- 审美敏感型：画面、配色、音乐、文案有一致品味。
- 事业/目标型：工作学习内容多，节奏感强。
- 情绪细腻型：深夜文案、歌词、天气、留白多。
- 社交中心型：朋友和活动多，重视体验和氛围。
- 慢热低展示型：动态少、隐私少、回应慢但稳定。

不要因为性别、穿搭或单张照片直接判断“好追/不好追”。

如果用户目标是追 ta，继续读取 `references/pursuit_playbooks.md`，把展示类型翻译成追法类型和拉扯强度。

### 3. 外貌、妆容、穿搭和审美线索

分析对方主动展示出来的外在风格和审美倾向。

可提取：

- 外貌展示方式：自拍多/他拍多/背影/局部/不露脸/合照。
- 妆容特点：淡妆、浓妆、清冷、甜妹、欧美、韩系、素颜感、口红色系等。
- 穿搭风格：运动、通勤、甜酷、极简、学院、辣妹、户外、复古等。
- 拍照风格：滤镜、构图、场景、是否重视氛围感。
- 审美偏好：咖啡、展览、live、书店、宠物、香水、花、运动、旅行。

用途：

- 夸具体审美，不空泛夸“好看”。
- 选择约会地点、礼物、花、聊天切入点。
- 帮用户展示同频的自己。

### 4. 互动方式

如果截图中有评论、点赞或互动，分析：

- 谁经常评论：朋友、同事、家人、异性、固定圈子。
- 对方回复评论的语气：热情、冷淡、玩梗、礼貌、撒娇。
- 互动习惯：是否公开暧昧、是否喜欢被起哄、是否回避隐私。
- 社交能量：外向热闹、熟人局、低调克制。
- 玩家信号：是否对多人使用相似暧昧语气，是否评论区热但私聊弱，是否用点赞/评论刷存在感。

不要把评论区一个人的玩笑当成事实。

遇到以下情况读取 `references/player_tactics_intel.md`：

- 朋友圈/小红书展示很强，但私聊很少推进。
- 评论区、点赞列表或合照里有多个暧昧对象线索。
- 节日前后突然发暧昧文案、歌、自拍或暗示。
- 文案像模板化恋爱话术，缺少具体对象。
- 用户降频后，对方不私聊，只通过动态/点赞刷存在感。

### 5. 微信息记忆

从朋友圈和聊天中提取可用于策略的微小信息：

| 微信息 | 记录方式 |
|--------|----------|
| MBTI | 只记“自称/提到”，不要当人格真理 |
| 星座/玄学 | 记录信不信、爱不爱聊、是否当梗 |
| 口头禅 | 具体词和出现情境 |
| 置信度 | ta说话是确定、犹豫、开玩笑、反复横跳 |
| 不一致/撒谎疑点 | 记录“前后不一致次数”，不要直接骂撒谎 |
| 心情变化 | 开心/累/emo/生气时的语气差异 |
| 礼物偏好 | 明确喜欢/讨厌的品牌、花、颜色、活动 |
| 忌口和生活习惯 | 饮食、作息、运动、过敏、宗教/文化禁忌 |
| 表情包/梗偏好 | 常用类型、含义、适合场景、避雷；未知梗先查证 |

写入 `profile.md` 时标注来源和置信度。

### 6. 可聊话题

从截图中提取 3 类话题：

- 轻话题：不容易冒犯，可以开场。
- 深一点的话题：适合已经聊过几轮后推进。
- 不建议碰的话题：隐私、身材、收入、前任、定位、过度评价外貌。

### 4. 展示自己策略

根据对方展示面，为用户设计“同频但不舔”的展示方式：

| 对方展示 | 用户可展示 |
|----------|------------|
| 旅行/探店 | 自己的城市发现、审美品味 |
| 健身/运动 | 自律、状态、轻松吐槽 |
| 工作学习 | 认真、靠谱、节奏感 |
| 情绪文案 | 稳定、能接情绪，但不急着拯救 |
| 朋友聚会 | 自己也有生活和社交，不围着对方转 |

### 5. 追法策略

根据朋友圈展示面输出：

- 追法类型：审美/仪式感型、情绪细腻型、社交活跃型、慢热低展示型、事业/目标型、高选择权/玩家型等。
- 拉扯强度：0-3。
- 适合的开场：夸审美/接情绪/展示生活/轻互怼/邀约测试。
- 适合的约会方向：咖啡、展览、散步、运动、live、轻餐、朋友局等。
- 用户该展示什么：生活状态、审美、幽默、工作节奏、社交温度。

### 6. 防文艺脑补

朋友圈文案、歌词、滤镜和氛围感只能说明展示倾向。不要因为文案深情、照片氛围暧昧、评论区玩梗，就判断对用户有高兴趣。

涉及“ta 对用户有没有意思”时，回到行动证据：

- ta 有没有私聊找用户。
- ta 有没有反问和补细节。
- ta 有没有推进见面或给替代时间。
- ta 有没有把公开互动转成私下明确安排。
- ta 有没有公开选择用户，而不是对多人释放相同信号。

## 输出格式

```markdown
## 朋友圈画像

**可见证据**
- {截图/文案中的具体元素}

**展示类型**：{1-2 个类型}
**可能性格/风格**：{可能偏向 + 证据}
**追法类型**：{类型名} · 拉扯强度 {0/1/2/3}
**外貌/妆容/穿搭线索**：{描述展示风格和审美倾向}
**互动方式**：{评论/点赞/回复语气，如有}
**近期玩家信号**：{无明显 / 有：平台人设型/评论区暧昧型/朋友圈定向投喂/节日借势等}
**微信息记忆**
- {MBTI/星座/口头禅/忌口/礼物偏好/心情差异等}
**表情包/梗偏好**
- {抽象/可爱-线条小狗/可爱-Chiikawa/帅气/长辈图反差/沙雕文字/不用表情包等 + 证据}

**聊天切入点**
1. {轻话题}
2. {深一点的话题}
3. {可轻撩的话题}

**不要碰**
- {容易油/冒犯/查岗/越界的话题}

**给用户的展示策略**
- {用户该展示什么}
- {怎么说得低油腻}

**可发开场/回复**
【稳妥版】{一句}
【会撩版】{一句，标油腻度}
【展示自己版】{一句}
```
