# 更新记录

## 5.3.0 · Temporal CNN — 2026-07-12

- 新增纯 TypeScript 时序卷积响应预测器（`decision/neural.ts`，1252 参数）：读取最近 45 天的观测时间线（9 维信号 + 观测密度通道，2.8 天分桶），两层一维卷积 + 全局平均池化 + 全连接，输出四类回应分布。前向与反向传播手写，经数值梯度校验；Adam、类平衡权重、标签平滑、L2；种子确定，回放可复现。
- CNN 只经诚实门控进入决策：按时间序留出最后 25% 真实结果做对照，保留集对数损失优于当时响应头 0.02 nats 才获得混合权重 min(0.5, n/(n+12))；混合在推演根节点逐分支线性实现，保持边际一致。数据不足或不更准时行为与 v5.2 完全一致。
- 每条真实结果都会触发重训（毫秒级、全本机），`rebuildDerivedState` 可完整重建；诊断接口新增 neuralPredictor 指标（样本、参数量、保留集对照、混合权重）。
- 决策网络图新增「时序卷积 CNN」节点：显示混合权重、参数量、训练样本与保留集优势，对各回应类别的推拉画成边。
- 离线评测新增 CNN 对照基准（`bun run evaluate`）：在「当前概要相同、轨迹形状不同」的合成序列上，CNN 保留集对数损失约 0.05 nats，概要结构头 1.61 nats。
- 测试 20 → 25：数值梯度校验、确定性收敛、基准优势、门控关闭、根节点混合方向。

详见 [v5.3.0 发行说明](docs/releases/v5.3.0.md)。

## 5.2.0 · Decision Network & Boldness — 2026-07-12

- 新增决策网络可视化：每份决策报告携带 `networkTrace`——引擎本轮真实计算的分层数据流（状态信念 → 体制假设 → 选中策略 → 动作特征 → 预测状态 → 回应分布 → 输出），右侧面板渲染为可交互 SVG，点节点查看激活值、参数与流入/流出贡献权重，边的粗细与颜色对应贡献强度和符号。
- 生成中的回复卡提供「看它的决策网络」入口，完成后追加「思考过程与决策网络」按钮；决策面板在流式输出开始前就已就绪。
- 顶栏新增两个滑杆：思考深度（快速/平衡/深入，控制 rollout 深度与体制分支数）与胆量（0–100，按档案持久化）。胆量重塑目标函数：风险惩罚 ×(1.45−0.9β)、信息价值 ×(1.2−0.4β)、追问成本 ×(0.6+0.8β)、弃权门槛 +0.14·2(β−½)，推进类动作获得带符号情境加成，探索奖励随胆量放大。
- 表达层随胆量调整语气：大胆档更主动直接，谨慎档动作小、留余地。
- 修复移动端抽屉遮罩层级：遮罩此前盖在侧栏之上，整屏发虚且侧栏点不到；现在侧栏浮在遮罩上方，遮罩只压暗不再模糊。
- 精简策略库：移除「说明边界」策略族；界面维度文案「边界敏感」改为「戒备程度」（内部键不变）。
- 文案清理：移除提示词与界面中自我设限的表述（不施压/不越界/不操控/不过度解读一类），观察器和表达层改为直接鼓励读透语气与潜台词；清除「不是X，而是Y」句式；节日与语气建议改为按对方情况取舍，不再默认追求方向。
- 测试 18 → 20：决策网络 trace 的结构与激活值、胆量对目标函数与 EVOI 成本的单调影响。

详见 [v5.2.0 发行说明](docs/releases/v5.2.0.md)。

## 5.1.0 · Learned World Model — 2026-07-12

- 世界模型从静态 (策略, 假设) → 奖励查表升级为真正的生成式模型：体制切换线性高斯动力学预测状态转移，协方差随动力学传播。
- 新增校准响应头：结构 softmax 与按（体制, 策略族）分桶、120 天半衰期衰减的 Dirichlet–多项经验计数做收缩混合。
- 模拟阶段改为信念空间 rollout：想象的回应经对角卡尔曼更新信念、经 Bayes 更新体制后验，根节点精确枚举回应、深层确定性等价延拓，深度随规划档位 1/2/3。
- 风险改为模型量：负面/失联概率加压力越界的高斯尾概率；模拟方差改为全方差分解（分支内回应随机性 + 分支间体制分歧）。
- “是否值得追问”改为按定义计算的 EVOI：枚举探针回应、Bayes 更新假设分布、重解最优单步值，替代原缺失计数启发式。
- 新增世界模型在线学习：真实结果同时更新响应计数、转移残差（带裁剪的 Robbins–Monro）与模型-基础频率对数损失门控；学习参数经收缩闸门生效，回放可完整重建。
- 决策证据检索升级为混合检索：Okapi BM25 + 特征哈希向量余弦 + 决策先验三路 Reciprocal Rank Fusion 融合，嵌入空间冗余抑制替代子串去重。
- 结构化 LLM 调用优先走 API 级约束解码（Anthropic 强制 tool schema、OpenAI 兼容 response_format），提示修复循环降级为回退路径。
- 诊断接口新增世界模型指标：已学习情境数、样本量、模型与基础频率的平均对数损失及优势。
- 测试从 11 个扩展到 18 个，覆盖体制动力学方向性、卡尔曼更新、EVOI 单调性、结果学习与混合检索。
- 重写 README 理论附录：POMDP 形式化、异方差遗忘估计、BM25/RRF、切换动力学、Dirichlet 收缩、卡尔曼与 Bayes 更新、有限视界回溯、EVOI、非平稳 bandit、不确定性泛函与校准指标全部落到公式。

## 4.0.0 · Adaptive Desktop — 2026-07-11

### 桌面发行

- 完整迁移到 Tauri 2 桌面应用，不再要求普通用户把仓库安装成 Skill。
- Tauri 启动编译后的 Bun sidecar，使用随机 localhost 端口；退出窗口时同时结束后端。
- 新增 GitHub Actions 发行矩阵：Windows NSIS/MSI、macOS Apple Silicon/Intel DMG、Linux x64/ARM64 AppImage 和 DEB。
- 安装包包含运行所需组件，用户不必安装 Bun、Node.js、Rust 或数据库。

### SQLite 长期记忆

- 新增 `memory.sqlite3`，使用 WAL、事务、索引和迁移表保存记忆卡、反馈事件和画像观察。
- 接入 `sqlite-vec` 0.1.9；系统允许原生扩展时使用 KNN 查询，否则自动回退到精确余弦检索。
- 原有 JSONL 素材记忆会自动导入 SQLite，同时保留 JSONL 作为可读恢复日志。

### 随时间变化的画像

- 每条可复制回复新增「记录后来结果」，可以记录回复内容、等待时间、是否继续聊、主动推进、兑现承诺和记得细节等信号。
- 用约 21 天与 240 天两个半衰期分别估计近期状态和长期习惯；证据自然衰减，避免旧印象永久统治判断。
- 新增变化检测：近期与长期有足够可信的偏离时显示「最近可能在变化」，不覆盖过去事实。
- 用有先验收缩的 Beta 后验学习短句、提问、玩笑、直接表达、邀约、关心等策略的实际效果；权重被限制在安全范围内。
- 文字表态和实际行动使用独立证据权重。兑现度下降时，兴趣判断会更重视行动。

### 易用性与安全

- 结果反馈采用一页人话表单，一次反馈不会给对方贴永久标签。
- 右侧判断面板显示已参考的真实结果、最近变化和较合适的表达方式。
- macOS 发布包优先保持签名与可启动性；无法安全加载原生向量扩展时透明降级。
- 重写 README、安装脚本、新手指南和 Codex 说明，全部以桌面产品为入口。

## 3.1.0 · Deep Memory — 2026-07-10

### 新功能

- 新建档案时可以一次选择任意数量的过往截图，不再设置固定张数上限。
- 浏览器逐个上传文件，服务端直接流式写入磁盘，避免把整批图片同时转成 Base64 堆在内存里。
- 新增可恢复的后台整理任务：截图由当前 AI 一张张读取，界面显示上传和分析进度；收起进度条、关闭弹窗或重新打开页面都不会丢失已创建的任务。
- 每张截图生成一张结构化记忆卡，包括摘要、可回指事实、人物、时间、情绪、重要度和检索描述。
- 新增本地 384 维向量索引、关键词重排和记忆关联。当前问题会从全部已整理截图中召回相关资料，旧资料不会仅因为时间久而被排除。
- 不支持看图或还在演示模式时，任务会安全停在「等待 AI」状态；切换到 Codex、Claude Code、Claude 或 GLM-4V 后自动继续。

### 易用性

- 新增全局批量进度卡，可收起、重试失败项、跳转到 AI 连接设置。
- 新建档案、AI 设置、移动端档案栏和判断面板均支持关闭按钮、点击背景和 `Esc` 退出。
- 批量选择时只渲染前 12 张预览，其余显示数量，避免数百张缩略图拖慢页面。
- 「本次判断依据」会显示素材库总量和本次语义召回数量。

### 修复

- Codex 额度或任务失败时优先显示结构化错误，不再让无关诊断警告遮住真正原因。
- 批量失败项可以原地重试，不需要重新上传原图。
- 后台任务状态和结果使用私有文件权限持久化，服务重启后仍能发现未完成任务。

### 技术说明

本版没有宣称真正的“无限内存”或“无损语义压缩”。上传数量不设产品上限，但仍受磁盘空间、所选模型额度和操作系统资源约束。完整原图和原始文字始终保留；向量和摘要只作为检索入口。

详见 [批量素材与长期记忆技术说明](docs/批量素材与长期记忆.md)。
