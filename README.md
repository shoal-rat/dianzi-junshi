追ta追到一半，不知道说什么。  
对象发来了一条说不清楚什么意思的消息。  
你想说某句话，但不确定该不该说。

**电子军师**是一个运行在 Claude Code 里的恋爱全程辅助工具。

从第一次聊天到稳定期——帮你读懂对方，帮你想清楚，帮你把对的话说对。

---

## 核心能力

### 一、关系阶段系统

不同阶段，策略完全不一样。

| 阶段 | 名称 | 建议基调 |
|------|------|---------|
| 0 | 初识期 | 有趣，不露底 |
| 1 | 暧昧期 | 若有若无，让ta猜 |
| 2 | 追求期 | 主动但有策略 |
| 3 | 告白/确认 | 清晰但不用力过猛 |
| 4 | 热恋初期 | 可以甜，但保留自我 |
| 5 | 稳定期 | 自然温暖，维持新鲜感 |
| 6 | 磨合期 | 降温，找核心问题 |
| 7 | 危机期 | 冷静克制，有尊严 |

### 二、油腻度控制

早期阶段最大的错误：**暴露需求感**。

"你最近是不是不想理我了"——这句话在暧昧期可能让你直接出局。

军师会根据当前阶段控制每条回复的"油腻度"：

- 初识/暧昧期上限 0–1.5/5：含蓄，有趣，不急
- 热恋期上限 3.5/5：可以甜，但甜而不腻
- 危机期上限 0.5/5：冷静，有尊严

### 三、三层消息解析

每一条消息，都有三层意思：

```
第一层  表面意思    他/她字面上说了什么
第二层  情绪状态    他/她现在真实的情绪是什么
第三层  真正需要    他/她真正想从你这里得到什么
```

"随便吧"可能是"我想让你帮我决定"。  
"没事你忙"可能是"我很在意但我不想先开口"。

### 四、军师模式

你有一句话想发给ta，但不确定——

```
/ask 我想说"你最近是不是忘了我"
```

军师帮你评估：能不能说、为什么、怎么改、什么时候发。

### 五、自动效果分析

发完消息后，上传后续聊天记录——

```
/upload-followup
```

军师自动分析ta的反应，判断这次策略是否有效，记录下来，下次更准。

### 六、情感价值驱动

回复的目标不只是"回应"，而是让对方产生真实的感受：

| 情感价值 | 意思 |
|----------|------|
| 被珍视 | ta感觉在你心里很重要 |
| 被逗乐 | ta真的笑了 |
| 被理解 | ta觉得你真的懂ta |
| 被撩到 | ta心跳加速了 |
| 被安心 | ta不再焦虑 |
| 被惊喜 | ta没想到你会这样说 |
| 被崇拜 | ta觉得你很有魅力 |

## 快速开始

```bash
git clone https://github.com/shoal-rat/reply-skill.git
cd reply-skill
```

在 Claude Code 中运行：

```
/zhunshi
```

回答 6 个问题（可跳过），建立档案，开始使用。

## 使用示例

### 场景一：暧昧期，她问"你最近忙什么呢"

```
/reply 你最近忙什么呢
```

**三层解读**

```
表面：询问你的近况
情绪：轻微焦虑/在意 · 强度 2/5
真正需要：被重视——她想知道自己在你生活里有没有分量
```

**回复方案（当前阶段：暧昧期 · 油腻上限 1.5/5）**

```
【方案 1】幽默 · 被逗乐 · 油腻度 0.5/5
————
"忙着活着，你呢"
————
预期反应：她可能觉得有点好笑，主动接话
契合度：8/10

【方案 2】含蓄暖心 · 被珍视 · 油腻度 1/5
————
"最近有点多，但有你们在还好"
————
预期反应：她注意到"你们"里有她，但又不确定，会想继续聊
契合度：9/10

【方案 3】制造好奇 · 被惊喜 · 油腻度 0.5/5
————
"有点事，之后跟你说，你最近怎么样"
————
预期反应：她好奇你在做什么，话题转到她身上
契合度：7/10
```

💡 推荐方案 2，在暧昧期把感情信号藏在日常里，效果比直说强。

```
/ask 我想问ta"你是不是有点喜欢我"
```

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  你想说："你是不是有点喜欢我"
  当前阶段：暧昧期（阶段 1）

结论：❌ 不建议

  阶段适配：3/10
  油腻度：3/5（上限 1.5/5）
  需求感风险：高

  风险：这句话把一个还在建立中的模糊情感
  变成了一个需要立刻回答的问题。
  对方如果还没想好，只能说不——
  即使ta实际上有感觉。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  【方案 1】制造场景让ta表态
  ————
  "上次那个地方你说想去，周末有空吗"
  ————
  为什么好：约见面比直接问更有效，见面后
  自然确认感情，主动方还是你

  【方案 2】暗示而不是问
  ————
  "最近聊天感觉你比以前话多了"
  ————

  ————
  [等ta有主动联系的信号后再推进]

  那时候你再推进成功率更高

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  现在不适合：关系还不稳，提前表态容易失去优势
```

## 命令速查

```
/zhunshi              建立新档案
/reply [消息]         分析并生成回复方案
/ask [我想说...]      军师评估你的想法
/analyze [消息]       仅分析，不生成回复
/upload-followup      上传后续聊天，分析效果
/stage [0-7]          更新关系阶段
/stats                查看策略效果统计
/update-partner       更新对象档案
/my-style             更新你的风格
/list-partners        查看所有档案
```

## 注意事项

- **隐私**：所有数据存储在本地 `partners/` 目录，不上传任何服务器
- **工具是工具**：军师帮你表达，但感情是真实的——不要只靠脚本
- **不操控**：所有建议的目标是真实的情感连接，不生成 PUA 类建议
- **你才是主角**：所有方案只是选项，你最了解你们的关系
