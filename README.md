## 它适合谁

你可能不是不会喜欢人，而是卡在这些地方：

- 看不懂 ta 的回复到底有没有意思。
- 想撩，但一不小心就油了、急了、暴露需求感。
- 对方忽冷忽热、只撩不约、画饼，你不知道该继续还是收。
- 约成了，但不知道订什么、送不送花、怎么确认细节。
- 同时聊几个人，怕记混，也怕每次都从头解释。

电子军师的定位很简单：**先展示你自己，再低油腻地撩；对方接球就推进，不接就收；识别到海王/海后，就换打法。**

## 核心能力

### 1. 看聊天，给能发的话

它会先拆 ta 那句话的字面、情绪和真实需求，再给三种回复：稳妥版、会撩版、展示自己版。每条都标油腻度，超出阶段上限会自动压回去。

### 2. 判断 ta 到底有没有意思

![兴趣度评分，加分减分证据，反舔狗模式止损](assets/demo-interest.svg)

主动找你、记得你说过的话、接得住你的梗、愿意见面，加分；长期只回「嗯嗯」「哈哈」、约了又拖、只在需要帮忙时冒头，减分。开了反舔狗模式，明显不值得加码时它会直接拉住你。

### 3. 反海王/海后：识别到就打擂台

![反海王海后模式：置信度、命中信号和拉扯策略](assets/demo-anti-player.svg)

它不只看老式的忽冷忽热，也看更近的平台化信号：模板化高情绪价值、评论区暧昧、朋友圈定向投喂、多线排班、节日借势、只在深夜回温。

海王/海后置信度偏高时，它会切到拉扯打擂台：

- **降频观察**：你不继续投喂，看 ta 会不会补主动。
- **轻推回球**：把“改天一定”推回到具体时间、地点、动作。
- **换时段测试**：避开 ta 固定回温时段，看投入是否还在。
- **专属细节测试**：漂亮话很多时，看 ta 记不记得你的小细节。
- **邀约兑现测试**：只说不做，就先不加码。

### 4. 不同类型，不同追法

直球低敏、慢热低展示、审美仪式感、社交活跃、高选择权玩家型，它会按聊天和朋友圈证据分型，再给拉扯强度。追男生和追女生的常见脚本也会校准，但不硬套性别。

### 5. 朋友圈和小红书式展示，它会看图

朋友圈一半信息都在图片里，不在文字里。截图丢进去，它会看妆容、穿搭、滤镜、构图、评论区互动、文案和表情包，提取话题、类型、仪式感偏好、约会地点和礼物线索。

### 6. 约到了，回复和旁白分开放

![左边是能发给对方的话，右边是只给你看的旁白](assets/demo-dateplan.svg)

可复制回复只放能发给 ta 的话；旁白提醒只给你看：哪天订位、送不送花、西餐怎么点、520/七夕/生日要不要提前准备。两栏分开，避免手滑误发。

### 7. 多对象长期记忆，不串戏

![每次打开先问你这次聊谁，可以选已有对象或新建一个](assets/demo-switch.svg)

每个对象一份独立档案：阶段、聊天习惯、哪招有效、海王/海后置信度、朋友圈画像、忌口、口头禅、表情包偏好，全分开记。换窗口也能接上。

## 输出会长什么样

遇到普通聊天，它给三条回复：

```text
稳妥版：刚忙完个项目，这两天在补觉。你呢
会撩版：忙着把日子过得有意思点，顺便等个有趣的人来问我
展示自己版：周末爬山去了，山顶风景绝了。你周末一般怎么过
```

遇到拉扯、低兴趣或海王/海后信号，它会额外给策略框，和可复制回复分开：

```text
┌ 策略
│ 回复间隔：1-4 小时
│ 消失建议：本轮收尾
│ 拉扯动作：轻推回球
│ 看反馈：ta 是否给时间地点，是否主动补细节
│ 回来反馈：发你实际发的版本、发送时间、ta 多久回、ta 回了什么
└
```

## 安装与上手

完全没用过也没关系，照着做。先挑一条适合你的路。

![三步：装一次、聊两句、发截图](assets/workflow.svg)

### 路线 A · Claude Code（推荐）

**第 1 步，装 Claude Code。**

- Windows：按 `Win + X`，点「终端」（老系统叫「Windows PowerShell」），粘这行回车：

  ```powershell
  irm https://claude.ai/install.ps1 | iex
  ```

- Mac：按 `Command + 空格`，打字「终端」回车打开，粘这行回车：

  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```

  装完按提示用 Claude 账号登录。

**第 2 步，把电子军师装成 Claude Skill。**

- Windows：

  ```powershell
  irm https://raw.githubusercontent.com/shoal-rat/dianzi-junshi/master/install.ps1 | iex
  ```

- Mac：

  ```bash
  curl -fsSL https://raw.githubusercontent.com/shoal-rat/dianzi-junshi/master/install.sh | bash
  ```

**第 3 步，开聊。** 输入 `claude` 回车，直接说「帮我追个人」。它会建档，然后你把微信截图丢进去。

> 每步长什么样、卡住怎么办，看[新手指南](docs/新手指南.md)。

### 路线 B · Codex

在 Codex 里，一行命令克隆到技能目录：

```powershell
git clone https://github.com/shoal-rat/dianzi-junshi.git "$HOME\.agents\skills\dianzi-junshi"
```

Mac / Linux 用：

```bash
git clone https://github.com/shoal-rat/dianzi-junshi.git "$HOME/.agents/skills/dianzi-junshi"
```

更多见 [platforms/codex.md](platforms/codex.md)。

## 平时怎么用

不用记命令，正常打字就行：

```text
（贴一张微信截图）这条我该怎么回
ta 这么说是什么意思，我还有戏吗
我想发「你是不是不想理我了」，行不行
感觉 ta 有点海后，帮我判断一下
ta 答应周末见面了，帮我安排一下
换个人，帮我新建个对象
```

习惯敲命令的话，`/reply`、`/interest`、`/moments`、`/date-plan`、`/anti-simp on` 这些它也认。

## 油腻度是什么

不是不让你甜，也不是不让你撩。油腻度只是提醒你：现在这段关系，信号给太满会不会把人吓跑。

| 阶段 | 大概上限 | 感觉 |
| --- | --- | --- |
| 初识 / 暧昧 | 0-1.5 / 5 | 有意思，但别露底 |
| 追求 / 确认 | 2-2.5 / 5 | 可以主动，别逼问 |
| 热恋 / 稳定 | 3-3.5 / 5 | 可以甜，别腻成复读机 |
| 磨合 / 危机 | 0.5-1.5 / 5 | 先降温，别上头 |

用户可见层会保留「油腻度」「会撩」「暧昧」「上头」「别露底」这些中文互联网语感，因为它们直观、有用。

## 运行原理

![从截图到三条回复的处理流程，外加越用越准的记忆回路](assets/demo-how.svg)

1. **读懂场景**：调出当前对象档案，看时间、阶段和最近进展。
2. **三层解读**：表面说了什么，是什么情绪，真正想要什么。
3. **盘算分寸**：有没有意思、是不是套路、当前能撩到几分。
4. **生成回复**：三条能发的话，再过一遍真人语气。
5. **写回记忆**：你发完反馈，它把有效策略、无效模式和对象偏好写回档案。

## 常见问题

**我电脑小白，连终端都没用过，能用吗？**
能。照路线 A 一步步粘命令就行，每条都给你写好了。

**粘命令报错，或者提示没有 git？**
Windows 先装 [Git](https://git-scm.com/downloads/win)，一路下一步，再回到安装步骤。

**装完它不理我，或者找不到？**
把 Claude Code 或 Codex 整个关掉再打开。新装的技能通常要重启才认得。

**怎么换对象？同时追好几个会不会乱？**
直接说「换人」或「新建」。每个人一份档案，各记各的。
