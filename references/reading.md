# 局势解读：消息、兴趣度、玩家信号

输入侧总纲：ta 发来的东西怎么读、有没有戏、是不是在养鱼。输出侧语感（替用户说的话）见 `references/voice.md`。**本文件是唯一权威**：反舔狗硬阈值、海王/海后置信度表都在这里，其他文件只引用、不复述。

## 目录

- 一、证据纪律：先破题名、原文与推断分开、行动证据优先
- 二、消息解读：先扫梗 → 三层解析 → 中文语气信号 → 消息类型 → 风险
- 三、兴趣度：四维评分、模仿度、加减分、反舔狗硬阈值、展示与轻撩
- 四、海王/海后：置信度系统、布线保线留鱼、常见模式、测试动作库
- 五、输出格式：消息解析 + 兴趣度判断

## 一、证据纪律

每个判断都要过这四关：

1. **原文与推断分开**。输出时"原文证据"和"推断"分栏写，不把"可能"说成"肯定"。证据不足就直说："这条信息不够判断，最稳妥的方向是……"
2. **先破题名**。用户给的代号、文件夹名、截图名只是标签，不是关系事实。档案叫"热恋记录"，也要独立校准：没承诺、没见面兑现、没对方主动推进，就直接写"证据不支持热恋"。
3. **行动证据优先**。甜话只算甜，行动才算数。是否主动找你、反问、给替代时间、推进见面、兑现承诺、公开选择你——这些才决定投入价值。
4. **文艺脑补降权**。深夜长文、歌单歌词、朋友圈文案、"很懂我""灵魂共鸣"体感、模板化温柔，只能提高"聊天甜度"，不能自动提高"关系承诺"和"行动兑现"。

### 证据来源标注

| 标记 | 含义 | 权重 |
|------|------|------|
| 原文证据 | 消息里直接出现的字词、标点、表情、时段 | 高，可直接引用 |
| 行动证据 | 主动找你、反问、给替代时间、推进见面、兑现、公开选择 | 最高，校准阶段和投入价值 |
| 档案记忆 | profile.md 高置信度记录（见 `references/memory.md`） | ●●● 强制应用 |
| 聊天模式 | 导入记录后的统计特征 | 风格校准用，不定性人格 |
| 朋友圈展示 | 截图里的文案、图片主题、频率（见 `references/partner.md`） | 话题和展示策略用 |
| 社区经验 | 中文互联网反复出现的恋爱观察 | 启发用，须回原文验证 |
| 情境信息 | 阶段、冲突、节日、深夜等 | 判断时机 |
| 标签信息 | 代号、文件夹名、项目名 | 只当索引 |

置信度：高=原文+档案或多次反馈一致；中=原文明确但缺验证；低=主要靠常见模式。

## 二、消息解读

### 先扫梗再解读

三层解析之前，先按 `references/memes.md` 的协议扫一遍：这条消息本身是不是梗？

- 「那咋了」「不知道，我的身材很曼妙」这类消息要按梗的语域读，不能按字面读。
- 把玩梗读成生气、把阴阳梗读成真夸，是分类失败模式（记为"把玩梗读成生气"，写进无效模式表）。
- 不认识的词先查 `references/glossary.md`，再查公开资料，查不到就标注"未知梗，按中性处理"。

### 三层解析

- **第一层 表面**：字面说了什么，发生了什么事。
- **第二层 情绪**：开心/分享欲 · 撒娇/求关注 · 难过/委屈 · 生气/不满 · 焦虑/不安 · 兴奋/期待 · 平淡日常 · 试探/测试 · 受伤/失落。
- **第三层 真正需要**：被看见 · 被夸/被认可 · 被陪伴 · 被哄 · 被安慰 · 被主动 · 被逗笑 · 被认真对待 · 要答案/建议。

解析前先标证据：关键词（称呼、情绪词、否定词）、语气（标点、表情、语气词）、节奏（长短句、连发、突然变冷/变热）、场景（时间、节日、刚吵完）、档案记忆、行动证据。

### 中文语气信号（对方发来的输入信号，不是你的输出规则）

| 信号 | 读法 |
|------|------|
| 单字「嗯」 | 冷或生气，红灯 |
| 「哦」 | 聊天终结者，基本聊死 |
| 「嗯嗯」「好哒」「哦哦」 | 正常甚至友好 |
| 从「好哒嗯嗯」退化到「嗯/哦」 | 降温红灯，优先级最高的信号 |
| 句号结尾 | 轻松语境里=情绪信号，可能不爽 |
| 省略号 | 犹豫、委屈、无语、欲言又止，看上下文 |
| 双字「哈哈」 | 冷淡缓冲，不是真笑 |
| 「哈哈哈哈哈」(5+)、「笑死」「绷不住了」 | 真笑 |
| 「随便吧/都行/看你」 | 可能是授权，也可能是隐性期待 |
| 表情包、微信内置表情 | 改变文字力度，部分内置表情有阴阳意味，看个人习惯 |
| 称呼变化（宝→名字→您） | 关系距离的实时读数 |

以上都是"这个人"的信号，优先看 ta 的历史基线，不按性别套模板。

### 消息类型

| 类型 | 特征 | 核心需求 |
|------|------|---------|
| 分享喜悦 | "你猜怎么了" | 一起高兴，被认同 |
| 日常问候 | "在干嘛" | 想联系，想知道你的状态 |
| 撒娇/卖萌 | 语气软、叠字、委屈表情 | 被哄，被夸 |
| 隐性抱怨 | "随便""没事""算了" | 要你主动追问和关心 |
| 显性抱怨 | "好烦""好累" | 被理解，不一定要解决方案 |
| 测试/试探 | "你觉得我…""你是不是…" | 要特定答案换安全感 |
| 情绪爆发 | "你干嘛""你怎么这样" | 先降温，再解决 |
| 深夜消息 | 没头没脑，深夜发 | 多想了，要陪伴和安心 |
| 发图/链接 | 分享内容 | 关注品味，一起讨论 |

### 风险评估

| 风险 | 说明 |
|------|------|
| 语言陷阱 | "你喜欢什么类型""你觉得她漂亮吗"——有标准答案 |
| 情绪地雷 | 明显不开心却说"没事"——不能信字面 |
| 期待落差 | ta 说了认真的事，敷衍会伤人 |
| 隐性需求 | 说的是 A，要的是 B |
| 时间敏感 | 深夜/节日/重要事件后，回复格外重要 |
| 冲突升级 | 指责、轻蔑、防御、冷处理苗头——先降温 |
| 压迫式要求 | 查岗、过度证明、隐私交换——收轻话题 |
| 套路/带节奏 | 忽冷忽热、画饼、只撩不约——见本文件第四节 |
| 行动缺席 | 聊得甜但不见面、不反问、不给时间——不能当高兴趣 |
| 结论性止步 | 有对象、只当朋友、明确拒绝——停止暧昧解读 |

## 三、兴趣度

### 四维评分

不给粗糙总分。先拆四维，再给总体判断和置信度：

| 维度 | 看什么 | 高分标准 |
|------|--------|----------|
| 聊天甜度 | 语气、暧昧、接梗、情绪价值、分享欲 | 会接话、会逗、会释放特别感 |
| 主动性 | 谁先找谁、谁开新话题、谁补细节 | ta 会主动找你，也会把话题带回你 |
| 关系承诺 | 身份、边界、公开选择、排他性 | 愿意把你放进真实关系位置 |
| 见面/行动兑现 | 邀约、替代时间、具体安排、事后补动作 | 给时间地点，拒绝也给替代方案 |

总体区间：0-2 明显没兴趣/已拒绝 → 降温或止损；3-4 礼貌低兴趣 → 少追问改展示自己；5-6 中性不稳定 → 低油腻轻撩测试；7-8 有兴趣 → 推进互动或邀约；9-10 双向奔赴 → 可升温但不油。

**维度覆盖规则**（总分由四维解释，不许互相掩盖）：

- 甜度不能覆盖行动失败。甜度 7、行动 1，只能判"会聊"，不能判"有稳定兴趣"。
- 主动性 <4 且用户连续主动 3 次以上，总体最多"中性不稳定"。
- 关系承诺 <3，不写"热恋""双向奔赴""确定关系"。
- 行动兑现 <3，下一步优先行动测试或降频观察，不继续堆情绪价值。
- 出现结论性证据（明确拒绝/有对象/只当朋友），停止暧昧解读，转止损或普通朋友策略。
- 单次回复慢不算证据，至少看齐四类。

### 模仿度=好感计

暧昧期聊天数据里，可测量的真实兴趣信号强度排序：

1. **语言模仿**：ta 开始复制你的语气词、句式、梗——最强信号。
2. **提问占比 >50%**：ta 的消息里一半以上在问你。
3. **表情包互换**：ta 用你发过的表情包回你，或你们形成共用包。

反向：单方面找话 + 各用各的表情包 = 敷衍。模仿度变化比单条消息热度更可信。

### 加分信号

| 信号 | 加分 |
|------|------|
| 主动开启话题、主动分享生活 | +2 |
| 会问你问题、提问占比高 | +2 |
| 记得你说过的细节 | +2 |
| 模仿你的梗或语气词 | +2 |
| 对邀约明确同意或给替代时间 | +2 |
| 回复有细节，不只是"嗯嗯/哈哈/好" | +1 |
| 接住你的梗或暧昧信号 | +1 |
| 表情包互换（用你的包回你） | +1 |
| 分享照片、日常、情绪、朋友圈内容 | +1 |
| 早期愿意持续多轮聊天 | +1 |

### 减分信号

| 信号 | 减分 |
|------|------|
| 明确说"不想谈/只当朋友/别误会" | -4 |
| 邀约总被拒且不给替代时间 | -3 |
| 长期只礼貌回复，不主动延续 | -2 |
| 多次只回"嗯/哦/哈哈/表情包" | -2 |
| 只在要帮忙、倒情绪、要资源时出现 | -2 |
| 对暧昧信号不接或转移话题 | -2 |
| 用户连续主动 3 次以上，ta 零主动 | -2 |
| 答完不反问、你找话 ta 不接、表情包各用各的 | -1 |

### 反舔狗硬阈值（全库唯一权威版本）

**没兴趣的强条件**（任一出现，明显降温）：明确拒绝或强调止步；已恋爱/有稳定对象；连续多次拒见面且无替代时间；只接受好处不给情感回应；你越主动 ta 越短越慢越礼貌。

| 触发 | 动作 |
|------|------|
| 连续 2 次不回且没有补解释 | 暂停主动 24-48 小时 |
| 连续 2 次你问完 ta 答完不反问 | 停止查户口，改展示自己或本轮收尾 |
| 连续 2 次邀约被挡且不给替代时间 | 停止邀约，降到普通朋友档 |
| 出现"我有对象了/别误会/只当朋友" | 停止暧昧解读，体面收尾 |
| 用户连续主动 3 次，ta 零主动开题 | 降频观察，看 ta 补不补主动 |
| 只在要资源、帮忙、倒情绪时出现 | 不主动投喂，转普通关系 |

反舔狗模式开启时，输出要像朋友拦你：清醒、直接、少安慰。证据够硬就直说"别聊暧昧了，这不是暧昧，是你在投喂"。不把"该断"包装成"再看看"：

```text
反舔狗模式判断：这段目前不值得继续加码。
理由：{2-3 个证据}
建议：停止追问和讨好，收回注意力，降到普通朋友档。
下一步：{一条体面收尾话 / 不发消息 / 换目标}
```

### 展示自己优先

总分 3-6 且无明确拒绝时，别查户口，先让用户展示自己。短、具体、有画面：

| 展示类型 | 内容 | 例子 |
|----------|------|------|
| 生活状态 | 正在做的事、运动、小成就 | 「刚跑完步 感觉又活了」 |
| 审美品味 | 音乐、店、电影、城市角落 | 「这家店的歌单还挺会选」 |
| 幽默感 | 自嘲、反差、轻吐槽 | 「今天脑子开省电模式了」 |
| 价值感 | 稳、靠谱、有自己的节奏 | 「手头的事先收个尾」「晚点看你发的这个」 |
| 社交温度 | 和朋友家人的自然互动 | 「刚被朋友拉去救场」「成功活着回来了」 |

### 低油腻轻撩测试

只给一个钩子，不连续追。ta 能接也能当普通话题过掉；接球再升温，不接就退回展示自己：

| 场景 | 轻撩 |
|------|------|
| ta 问你最近忙什么 | 「在憋一个大动作」「等成了第一个讲给你听」 |
| ta 发自拍/穿搭 | 「这个状态有点犯规」「我先假装冷静」 |
| ta 说累 | 「那今天先不当大人了」「给你五分钟摆烂额度」 |
| ta 夸你 | 「你这样说我会短暂膨胀一下」 |

追法类型、拉扯强度、回复间隔、气质匹配的具体打法见 `references/strategy.md`。

## 四、海王/海后

置信度是对"玩家型/养鱼型互动"的概率揣测，不写死结论。判断先看聊天证据和历史行为，社区经验只做校准。

### 置信度档位

| 分数 | 档位 | 含义 |
|------|------|------|
| 0-20 | 很低 | 更像慢热、忙、或聊天习惯问题 |
| 21-40 | 偏低 | 有一点玩家信号，证据弱 |
| 41-60 | 中等 | 多个信号，需要降频观察和邀约测试 |
| 61-80 | 偏高 | 很像养鱼/只撩不推进，拿回节奏 |
| 81-100 | 很高 | 连续模式明显，反舔狗模式下直接止损或换目标 |

### 加分信号（唯一权威版本）

| 信号 | 加分 |
|------|------|
| 朋友圈/评论区/点赞出现多个暧昧对象线索 | +20 |
| 长期只撩不约，聊天亲密但现实安排缺席 | +15 |
| 释放暧昧但不给关系定位或社交认证 | +15 |
| 你降温 ta 回温，你热起来 ta 又冷 | +15 |
| 画未来但没有兑现动作 | +15 |
| 平台展示很强，私聊/见面推进连续缺席 | +15 |
| 固定时段回温，像多线排班 | +15 |
| 关系/见面问题总被转成情绪话题 | +15 |
| 深夜热白天冷，或只在酒后/emo 时强烈暧昧 | +10 |
| 前后说法不一致，且和见面/关系状态有关 | +10 |
| 只在要资源、情绪价值、帮忙时出现 | +10 |
| 给所有人同样温暖，你没有特殊感 | +10 |
| 模板化高情绪价值，连续缺少你的专属细节 | +10 |
| 你降频后用公开动态或点赞刷存在感 | +10 |
| 节日前后突然强暧昧，但不安排具体行动 | +10 |

### 降分信号（唯一权威版本）

| 信号 | 降分 |
|------|------|
| 主动给明确时间地点并兑现 | -20 |
| 拒绝时给替代时间 | -15 |
| 愿意让你进入真实生活圈 | -15 |
| 公开暧昧能转成私下明确安排 | -15 |
| 忙完主动补解释、补行动、补时间 | -15 |
| 节日前后有实际安排、时间、地点或投入 | -15 |
| 对你有独特投入，不是群发式温暖 | -10 |
| 能说出并连续记得你的专属细节 | -10 |
| 会解释消失原因，且行为能对上 | -10 |
| 朋友圈/评论区互动和私聊逻辑一致 | -10 |

### 布线 → 保线 → 留鱼（玩家式经营三阶段）

- **布线**（建多条线）：快速加微信但不推进现实；动态持续发可多人解读的暧昧素材；评论区固定出现多个暧昧对象；对不同人用相似昵称和开场；回复节奏像排班。
- **保线**（低成本维持）：周期性回温——你冷了来一句，你热了又退；早安晚安/想你/宝宝等亲密词多但不落到见面；"忙/家里有事/过两天细说"反复出现但缺细节；"下次一定/等我忙完"长期无日期；不介绍朋友、不公开同框。
- **留鱼**（防你跑掉）：你要撤时突然变甜变脆弱；提关系/见面就转成情绪话题或未来想象；你不主动后用点赞评论刷存在感；用"你是不是不在乎我了"把主动权推回你；只在深夜、酒后、emo 时给强暧昧。

### 常见模式

| 模式 | 表现 | 判断点/策略 |
|------|------|------------|
| 正常拉扯 | 有来有回，有暧昧也有行动 | 陪着拉，保留余味 |
| 慢热 | 回复慢但稳定，细节和见面会兑现 | 稳定展示自己，少逼问 |
| 忙但有兴趣 | 时间少，但补解释、补问题 | 不加戏，按节奏接 |
| 养鱼 | 亲密词多行动少，多对象痕迹 | 降频观察，轻推回球 |
| 画饼 | 未来很多，具体安排没有 | 要具体时间地点，不接空头承诺 |
| 回温卡点 | 你撤退才热，你投入就冷 | 接住不加码，降频观察 12-48h |
| 资源触发 | 有事才找，没事消失 | 不主动投喂，转普通关系 |
| 平台人设 | 动态精致，私聊推进弱 | 夸具体审美后低成本邀约测试 |
| 多线排班 | 固定时段回温，内容浅 | 换时段测试，减少投喂 |
| 模板情绪价值 | 话术甜且完整，缺你的专属细节 | 专属细节测试，看记忆和行动 |

### 测试动作库

写进策略框或旁白，不放进可复制回复：

- **专属细节测试**：提一个只有认真聊过才记得的小信息，看 ta 接不接得上。
- **低成本邀约测试**：轻量地点 + 两个时间选项，看 ta 给不给明确反馈。
- **换时段测试**：避开 ta 固定热络时段，看是否仍愿意投入。
- **私聊兑现测试**：对朋友圈/评论区暧昧轻推，看 ta 转不转私聊行动。
- **节日兑现测试**：节点前看 ta 是只口头暧昧，还是给安排。
- **降频观察**：连续主动后暂停 12-48 小时，看 ta 补不补主动。
