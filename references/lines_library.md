# 话术素材库 + 自带话术校准

这个文件干两件事：

1. 给一批**原创**的、按场景 / 气质 / 阶段分好、并标好油腻度的话术素材，供生成回复时取用和改写。
2. 给一套机制：用户从任何地方看到、想用的话术，丢进来，由军师按当前对象的阶段、气质和油腻度上限**校准**后再用。

两条铁律：

- **不照搬任何具体博主、视频或他人作品的成段文案。** 看到喜欢的风格可以学"为什么好笑、为什么不油"，但落到输出一律现写。用户自带的句子按下面第三节校准，不整段复制粘贴别人的版权内容。
- 素材只是骨架，最终一定过 `prompts/human_voice.md`（去机器腔）、`prompts/oil_control.md`（控油）、`references/persona_modes.md`（对气质），改成像用户本人会发的话。直接照搬素材库会显得是模板。

---

## 一、什么样的话术不油

油不油不取决于甜不甜，取决于这几点：

- **挂在事上，不悬空**："你这个歌单品味可以啊" > "你好有品味"。具体的才可信。
- **留白，不交底**：点到为止，让对方接一句，而不是把好感一次倒完。
- **有自己的视角，不只夸对方**："这家店我找了挺久" 比 "你喜欢就好" 有人味。
- **轻，不用力**：玩笑、反差、顺口一提，比郑重其事的告白式表达更耐听。
- **不暴露需求感**：不问"你怎么还不回"，不解释自己一长串，不追问。

油的几个典型来源（生成时自查）：把对方架太高的夸、硬凹的深情、用力的称呼、查岗式提问、把对方的礼貌当回应继续加码。

## 二、原创素材（按场景）

每条标注：`油腻度 N/5` 和适合气质 `[痞/乖/通用]`。实际使用时按阶段上限压回去，按用户风格改写。

### 开场 / 破冰

- `[通用 0.5]` 顺着对方朋友圈一个具体细节起话：`你那张图的店在哪，看着不错`
- `[痞 1]` `我猜你是那种朋友圈很会拍、本人更难搞的`
- `[乖 1]` `刚刷到个东西想到你说过的那件事，就来问问`

### 接住情绪（ta 说累 / emo）

- `[乖 1]` `先别硬撑，喝口水歇会儿，想说就说，不想说我陪着`
- `[痞 1]` `这么累还回我消息，行，记你一功`
- `[通用 1]` `今天是被生活摁在地上摩擦的一天哈，晚点请你吃个好的回回血`

### 轻撩 / 留余味（暧昧期，注意阶段上限 1.5）

- `[通用 1]` `跟你聊天有点上头，得控制一下`
- `[痞 1.5]` `你这人不太好搞定，但我有耐心`
- `[乖 1]` `不知道为什么，跟你说话特别放松`
- 收尾留钩：`先去忙了，剩下半个故事下次讲`

### 邀约（落到具体，别悬在"改天"）

- `[通用 1]` `周六下午有个展，顺路，要不要一起去看看`
- `[痞 1]` `给你个机会请我喝杯咖啡，这周` （仅对吃痞、关系够的对象）
- `[乖 1]` `那家你想去的店我查了下，周末人少，去不去`
- 被挡时不加码：`行，那等你有空，不急`

### 拉扯 / 回温（配合 `references/tactics_and_pushpull.md`）

- 对方久不回、回来时：`[痞 1] 哟，想起我了`
- 对方忽冷忽热：不接空话，`[通用 1] 光说不练可不行，定个时间`
- 自然展示自己：`[通用 0.5] 刚爬完山，腿废了，但山顶是真的绝`

### 约会后跟进

- `[通用 1]` `今天挺开心，下次带你去那家我说的店`
- `[乖 1]` `到家跟我说一声`
- `[痞 1]` `今天表现不错，下次继续`

## 三、自带话术校准（用户从别处看来的句子）

用户经常会从短视频、评论区、朋友那儿看到一句觉得"好会说"的话，想用。机制是**校准**，不是照抄：

1. 用户把句子丢进来（"我看到一句'……'，能用吗"）。
2. 军师按以下维度判断和改写：
   - **油腻度**：现在几分，超没超当前阶段上限；超了就压。
   - **阶段适配**：这句在你俩当前阶段会不会太前或太冒犯。
   - **气质适配**：对方吃痞还是吃乖，这句要不要换个味道（见 `persona_modes.md`）。
   - **像不像用户本人**：套话、网感太重、用户平时不会这么说的，改成 ta 的口气（见 `style_calibrator.md`）。
   - **真实性**：别让用户背一句自己都不信、现场圆不回来的台词。
3. 输出：能用 / 改一下能用 / 不建议用，给出改写版和一句理由。

```markdown
**你想用的**："{原句}"
**判断**：{能用 / 改一下能用 / 不建议}
**油腻度**：{N}/5（当前阶段上限 {M}/5）
**改写版**："{贴合阶段、气质、你本人口气的版本}"
**为什么**：{一句}
```

> 关于"想学某个博主的话术风格"：可以学风格（为什么不油、节奏怎么留白），但请把你具体想用的句子贴进来按上面校准，军师不抓取、不整段搬运任何博主或视频的原文，避免照搬版权内容，也避免你背一段不像自己的词。

---

## 四、使用规则

- 素材是骨架，不是成品；一律过控油、去机器腔、对气质、像本人四道再输出。
- 可复制回复默认只写文字，不混表情包建议（表情包另写，见 `references/sticker_pack_guide.md`）。
- 同一意图尽量给不同气质 / 不同油腻度的版本，让用户挑。
- 一条回复宁可短、留白，也不要为了用素材而把话说满。
- 油腻度永远服从当前阶段上限（见 `prompts/oil_control.md`），痞不是更油的借口。
