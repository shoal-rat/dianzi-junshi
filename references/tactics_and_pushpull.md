# 拉扯打法 + 海王/海后置信度

这个文件管三件事：

1. 给用户制定暧昧期拉扯策略。
2. 判断对方是不是在用海王/海后式养鱼打法。
3. 把“多久回、要不要短暂停顿、下一步看什么反馈”写成策略方框。

判断先看聊天证据和历史行为，社区经验只做校准。

## 拉扯的可用动作

拉扯的核心是控制节奏：给信号、留余味、等对方补动作。

常用动作：

- **延迟回复**：不是不回，而是把回复间隔拉到符合用户生活节奏的位置。
- **短暂停顿**：本轮不继续追问，等对方下一次主动或等新话题出现。
- **留钩子**：说一半，留下对方会想问的点。
- **轻推回球**：对方只撩不推进时，把选择权推回去，看 ta 有没有实际动作。
- **展示自己**：发一个自己的状态、审美、社交或工作生活片段，减少围着对方转的感觉。
- **降频观察**：连续低质量回应时，减少主动开局次数，看对方会不会补主动。
- **反差称呼/语气**：昵称和正经称呼切换，制造一点距离变化。
- **邀约测试**：聊热后给一个低成本见面选项，看 ta 是否给时间、地点或替代方案。

## 回复间隔建议

按场景给具体时间，不写空泛的“晚点回”。

| 场景 | 建议间隔 | 说明 |
| --- | --- | --- |
| 对方高投入、正在热聊 | 看到后 1-10 分钟 | 保持连续性，别突然断 |
| 对方普通日常问候 | 10-40 分钟 | 像正常生活里抽空回 |
| 对方轻撩但没推进 | 20-90 分钟 | 给一点余味，回得太快容易显急 |
| 对方只回短句/表情包 | 1-4 小时或本轮收尾 | 不继续追问，等新话题 |
| 对方忽冷忽热后回温 | 30 分钟-3 小时 | 接住但不立刻加码 |
| 对方画饼/只撩不约 | 本轮轻推回球；之后等 ta 补动作 | 重点看行动 |
| 深夜对方情绪上来 | 视关系阶段，5-20 分钟 | 如果用户也在线，可以短句接住 |
| 用户已经连续主动 2-3 次 | 暂停 12-48 小时 | 等对方主动或等真实新素材再开 |

## 要不要“消失一段时间”

输出时直接给结论：`不建议消失 / 建议短暂停顿 / 建议本轮收尾 / 建议 12-48 小时不主动`。

判断：

- 对方热聊中：不建议消失。
- 对方刚给出高质量回复：不建议消失，应该接住。
- 对方冷淡但没明确拒绝：建议短暂停顿或本轮收尾。
- 用户连续主动、对方连续低质量：建议 12-48 小时不主动。
- 对方只在用户要走时回温：建议接一下但不加码，之后等 ta 补行动。
- 对方已明确给替代时间或实际安排：不建议消失，推进约定。

## 策略方框

每次 `/reply` 或 `/interest` 涉及推进、降温、拉扯、反舔狗时，都加这个框。框只写策略，不放可复制回复。

```text
┌ 策略
│ 回复间隔：{现在回 / 10-40 分钟 / 1-4 小时 / 今晚不主动 / 12-48 小时不主动}
│ 消失建议：{不建议消失 / 短暂停顿 / 本轮收尾 / 暂停主动}
│ 拉扯动作：{留钩子 / 轻推回球 / 展示自己 / 降频观察 / 邀约测试}
│ 看反馈：{ta 是否反问 / 是否补细节 / 是否主动开新话题 / 是否给时间地点}
│ 回来反馈：发你实际发的版本、发送时间、ta 多久回、ta 回了什么
└
```

## 海王/海后置信度系统

输出“海王/海后置信度”，不要写死结论。分数是对“玩家型/养鱼型互动”的概率揣测。

| 分数 | 档位 | 含义 |
| --- | --- | --- |
| 0-20 | 很低 | 当前更像普通慢热、忙、或聊天习惯问题 |
| 21-40 | 偏低 | 有一点玩家信号，但证据弱 |
| 41-60 | 中等 | 多个信号出现，需要降频观察和邀约测试 |
| 61-80 | 偏高 | 很像养鱼/只撩不推进，建议拿回节奏 |
| 81-100 | 很高 | 连续模式明显，反舔狗模式下直接止损或换目标 |

### 加分信号

| 信号 | 加分 |
| --- | --- |
| 长期只撩不约，约到实际时间就模糊 | +15 |
| 同时释放暧昧但不给关系定位或社交认证 | +15 |
| 深夜热、白天冷，且反复出现 | +10 |
| 用户降温时 ta 回温，用户热起来 ta 又冷 | +15 |
| 画未来但没有兑现动作 | +15 |
| 聊天亲密，现实安排总是缺席 | +15 |
| 朋友圈/社交圈里明显有多个暧昧对象线索 | +20 |
| 前后说法不一致，且和见面/关系状态有关 | +10 |
| 只在需要资源、情绪价值、帮忙时出现 | +10 |
| 给所有人同样温暖，用户没有特殊感 | +10 |

### 降分信号

| 信号 | 降分 |
| --- | --- |
| 主动给明确时间地点并兑现 | -20 |
| 愿意让用户进入真实生活圈 | -15 |
| 会解释消失原因，且行为能对上 | -10 |
| 拒绝时给替代时间 | -15 |
| 对用户有独特投入，不只是群发式温暖 | -10 |

### 输出格式

```markdown
**海王/海后置信度**：{N}/100（{很低/偏低/中等/偏高/很高}，置信度 {高/中/低}）
- 加分证据：{最多 3 条}
- 降分证据：{最多 2 条}
- 下一步测试：{邀约测试/降频观察/轻推回球/等对方补主动}
```

## 常见模式

| 模式 | 表现 | 策略 |
| --- | --- | --- |
| 正常拉扯 | 有来有回，有暧昧，也有行动 | 陪着拉，保留余味 |
| 慢热 | 回复慢但稳定，见面/细节会兑现 | 稳定展示自己，降低逼问 |
| 忙但有兴趣 | 时间少，但会补解释、补问题 | 不加戏，按节奏接 |
| 养鱼 | 亲密词多，行动少，多个对象痕迹 | 降频观察，轻推回球 |
| 画饼 | 未来很多，具体安排没有 | 要具体时间地点，不接空头承诺 |
| 回温卡点 | 用户撤退时突然热，过后又冷 | 接住但不加码，看后续兑现 |
| 资源型出现 | 有事才找，没事消失 | 不主动投喂，转成普通关系 |
