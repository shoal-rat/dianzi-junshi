## 一、阶段与油腻度上限

| 阶段 | 名称 | 油腻上限 | 基调 |
| --- | --- | --- | --- |
| 0 | 初识期 | 0/5 | 普通朋友语气，目标是「有意思」不是「喜欢你」；回复频率对等或稍低 |
| 1 | 暧昧期 | 1.5/5 | 若有若无，让对方猜；绝不暴露「我在等你回」 |
| 2 | 追求期 | 2/5 | 好感可以明显但留余地；重点制造见面，邀约要有具体理由 |
| 3 | 告白/确认 | 2.5/5 | 清晰但不用力过猛，像说一件自然的事；对方没立刻答应不追问 |
| 4 | 热恋初期 | 3.5/5 | 可以甜，保留独立和不可预期性 |
| 5 | 稳定期 | 3/5 | 自然舒适 > 刻意甜；新鲜感靠制造小惊喜 |
| 6 | 磨合期 | 1.5/5 | 降温为主；说感受（「我有点难受」）不翻旧账（「你总是」） |
| 7 | 危机期 | 0.5/5 | 冷静有尊严；不解释轰炸、不挽留轰炸，重要的话短而有力 |

阶段判断和升降级的行动证据规则在 `references/reading.md`（先破题名 + 行动证据校准）。要点一句话：**聊得再甜也升不了阶段，时间、选择权和行动放到用户身上才算数。**

早期（0-2）红线：不发「暗示我在等你」的话；对方一条你别回三条以上；深夜别先开局（对话进行中除外）；对方连续冷淡时不加码。

## 二、油腻度怎么打分（先说理由再给分）

油腻度 = 亲密压力 / 需求感暴露程度，0-5：

| 分 | 感觉 | 例 |
| --- | --- | --- |
| 0 | 无情感信号 | 「这店周末人多吗」 |
| 1 | 极含蓄，若有若无 | 「你今天好像有点蔫」 |
| 2 | 有明显好感不腻 | 「跟你聊天挺上头的」 |
| 3 | 直接的情感表达 | 「想你了」 |
| 4 | 高浓度爱意 | 「宝 今天怎么样」 |
| 5 | 高压黏着 | 「你为什么不回我 我一直在等」 |

**打分程序（防自我放水）**：先用一句话描述「这句暴露了多少需求感、给对方多大表态压力」，再根据描述给分——不许先写个正好卡在上限的分数再找理由。拿不准时往高了打。生成的方案超上限就删掉重写，不是把标注改小。

需求感的隐蔽形态（各 +1）：等待感（「怎么现在才回」）、存在感焦虑（「你是不是不想理我了」）、连问两遍同一问题、发完「好的」又补「那就这样」、回复长度超对方 3 倍、句尾挂个不是问题的「？」。

反需求感 = 把注意力放回自己的生活：说自己正在做的事顺带聊、你先结束对话、分享状态而不是查对方状态。

## 三、拉扯动作与回复间隔

可用动作：**留钩子**（说一半，留对方想问的点）、**轻推回球**（对方只撩不推进时把选择权推回去：「行啊 那定哪天」）、**展示自己**（发自己的状态/审美/生活片段）、**降频观察**（连续低质量回应时减少主动，看 ta 补不补）、**本轮收尾**（不追问，等新话题）、**邀约测试**（聊热后给低成本见面选项，看给不给时间）。

| 场景 | 回复间隔 |
| --- | --- |
| 对方高投入热聊中 | 1-10 分钟，别突然断 |
| 普通日常问候 | 10-40 分钟，像生活里抽空回 |
| 对方轻撩但没推进 | 20-90 分钟，留点余味 |
| 对方只回短句/表情包 | 1-4 小时或本轮收尾 |
| 忽冷忽热后回温 | 30 分钟-3 小时，接住不加码 |
| 画饼/只撩不约 | 本轮轻推回球，之后等 ta 补动作 |
| 用户已连续主动 2-3 次 | 暂停 12-48 小时，等 ta 主动 |

要不要停，直接给结论：不建议消失 / 短暂停顿 / 本轮收尾 / 12-48 小时不主动。热聊中和对方刚给高质量回复时永远不建议消失；对方给了明确时间地点就推进安排，别继续玩节奏。

涉及推进、拉扯、降温、低兴趣时输出策略方框（只写策略，不放可复制回复）：

```text
┌ 策略
│ 回复间隔：{现在回 / 10-40 分钟 / 1-4 小时 / 今晚不主动 / 12-48 小时不主动}
│ 消失建议：{不建议消失 / 短暂停顿 / 本轮收尾 / 暂停主动}
│ 拉扯动作：{留钩子 / 轻推回球 / 展示自己 / 降频观察 / 邀约测试}
│ 看反馈：{ta 是否反问 / 是否补细节 / 是否主动开新话题 / 是否给时间地点}
│ 回来反馈：发你实际发的版本、发送时间、ta 多久回、ta 回了什么
└
```

**无效拉扯警告**：拉扯是筛选和节奏工具，不是欲擒故纵表演。当下年轻人明确反感「为拉而拉」——忽冷忽热折磨人不是魅力，是让人累。对直球、慢热、认真型对象硬拉扯，会把本来有戏的线弄断。

## 四、拉扯强度 0-3

| 档 | 名称 | 用法 | 适用 |
| --- | --- | --- | --- |
| 0 | 稳定接球 | 看到就回、认真接、少玩节奏 | 明确高兴趣、慢热稳定、直球型 |
| 1 | 轻拉扯 | 10-40 分钟回、留钩子、轻推回球 | 普通暧昧、接梗但不够主动 |
| 2 | 中拉扯 | 1-4 小时回、本轮收尾、等对方补动作 | 高选择权、社交活跃、忽冷忽热 |
| 3 | 高拉扯 | 12-48 小时不主动、只奖励高质量回应、行动测试 | 玩家信号偏高、只撩不约、用户已连续主动 |

高拉扯（2-3）要满足 2 条以上才用：对方选择多对热情免疫 / 接暧昧但推进模糊 / 用户已连续主动、对方只短回 / 用户降频时对方会回温 / 海王置信度 41+。执行：高质量回应正常接住，低质量不追问本轮收尾，只撩不约就轻推回球，回温不兑现就接住不加码。

低拉扯保护名单（硬拉扯会弄断的）：直球型、慢热稳定型、认真目标型、情绪细腻型、已明确给时间地点或持续主动的人。对他们：稳定接球 + 低油腻轻撩 + 具体邀约。

## 五、追法类型

先看聊天证据，再看朋友圈展示，性别脚本只在互动模式明显时才套，用户反馈随时修正。

**追女生常见类型**：

| 类型 | 可见证据 | 主策略 | 强度 |
| --- | --- | --- | --- |
| 审美/仪式感型 | 朋友圈重滤镜、穿搭、咖啡、展览 | 夸具体审美，提前设计约会体验 | 1-2 |
| 情绪细腻型 | 文案多、歌词、深夜、讲感受 | 先接情绪，再轻轻展示稳定感 | 0-1 |
| 社交活跃型 | 聚会多、评论区热闹 | 展示自己的生活圈，约有氛围的体验 | 1-2 |
| 慢热低展示型 | 动态少、回复慢但不敷衍 | 降低频率，靠具体细节慢慢升温 | 0-1 |
| 事业/目标型 | 工作学习、健身、自律 | 短句有信息量，邀约高效率 | 0-1 |
| 高选择权/玩家型 | 亲密词多、现实动作少 | 少投喂情绪价值，轻推回球，行动测试 | 2-3 |

误区：查户口式提问、太早站到追求者位置被高标准审视、只给情绪价值不展示自己、聊得热不落到见面。

**追男生常见类型**：

| 类型 | 可见证据 | 主策略 | 强度 |
| --- | --- | --- | --- |
| 直球低敏型 | 读不太懂暗示，回复朴素稳定 | 信号给清楚点，别让他猜 | 0-1 |
| 事业/理性型 | 目标感强、话少有逻辑 | 具体轻松高效率，少情绪长文 | 0-1 |
| 幽默游戏型 | 爱玩梗、互怼、游戏运动 | 挑战感、轻互怼、反差称呼 | 1-2 |
| 慢热被动型 | 会回但少主动，等球 | 明确开口留台阶，给低成本邀约 | 0-1 |
| 高自尊/高选择权型 | 被夸惯、选择多 | 少捧多筛选，夸具体不追着确认 | 2 |
| 玩家型 | 很会撩、行动少、深夜出现 | 不接空话只看兑现，降频观察 | 2-3 |

误区：只给暗示期待自动读懂、过度端着错过窗口、把「回得直」误判成没兴趣、对玩家型给太多情绪反馈变成备选。

输出块（`/interest`、`/moments`、复杂 `/reply` 需要时）：

```markdown
**追法类型**：{类型} · 拉扯强度 {0-3}
**为什么**：{2 条可见证据}
**打法**：{展示自己 / 轻撩 / 降频观察 / 轻推回球 / 邀约测试}
```

## 六、气质频道：痞一点 / 乖一点

不是二选一，是一条带子。同一句话换个气质说，效果差一倍。

| | 痞 / 桀骜 | 乖 / 温柔 |
| --- | --- | --- |
| 给人的感觉 | 有自己的节奏、不轻易交底、带挑战和反差 | 稳、可靠、让人安心 |
| 典型动作 | 轻怼、留白、不秒回、敢开玩笑敢拒绝 | 认真接话、记得细节、主动安排、情绪稳 |
| 撩法 | 点到为止、反差和余味 | 真诚直接、在乎和陪伴 |
| 翻车成 | 油、装、爹味、忽冷忽热让人累 | 舔、闷、没火花、老好人 |
| 谁吃这套 | 选择多、容易腻、喜欢被挑战、自己带反骨 | 求稳、要安全感、被撩怕了、慢热 |

判断对方吃哪种，看证据：ta 自己说话带刺爱互怼、对周到热情免疫、喜欢「不好搞定」的角色 → 偏吃痞；慢热、被撩会紧张、在乎「记不记得我说的事」、你一玩节奏 ta 就缩 → 偏吃乖。分情境的很常见（正经事要你靠谱，打闹时吃你痞），写清楚什么场合走哪种。证据不足默认偏乖。

**只放大用户真实有的一面**。完全不会怼人的用户，不硬塞痞话术，给「少回一句、留个白」的轻量版。痞不等于可以更油，乖不等于可以舔，油腻度始终服从阶段上限。

同一意图的两个频道（例：对方说「我今天好累」）：

- 痞向：`这么累还回我 感动住了`（轻调侃留余味）
- 乖向：`累成这样` + `那先瘫着 想吐槽随时说`（稳、在乎、不索取）

例：对方久不回，回来了：

- 痞向：`稀客啊`
- 乖向：不提这事，用新话题自然起：`刚刷到个东西想到你`

## 七、性格底色

比阶段和兴趣更慢的一层，解释「为什么 ta 会这样反应」。全部基于证据，不足标 `[待观察]`，不靠星座 MBTI 硬填。

| 轴 | 两端 | 怎么看出来 |
| --- | --- | --- |
| 社交能量 | 内向单线深聊 ↔ 外向人群充电 | 朋友圈社交密度、约会偏好热闹还是安静 |
| 求稳↔求新 | 熟悉可预期 ↔ 新鲜爱尝试 | 对没试过的事和临时变化的反应 |
| 理性↔情绪 | 先讲逻辑 ↔ 先讲感受 | 倾诉时要建议还是要共鸣 |
| 计划↔随性 | 提前定 ↔ 临时起意 | 约会要不要提前敲细节 |
| 守规则↔挑战感 | 按规矩 ↔ 爱一点反差叛逆 | 决定 ta 吃乖还是吃痞（接第六节） |
| 表达↔克制 | 直说 ↔ 收着要被读懂 | 潜台词多不多 |
| 安全感来源 | 掌控 / 被在乎 / 独立空间 / 被认可 | ta 什么时候最放松、什么时候多想 |

配套两张清单（记进档案，见 `references/partner.md`）：**喜欢**（眼睛会亮的具体事）、**不喜欢**（兴致会掉的）、**想要没说透**、**想做不敢做**（ta 流露过想试没试的小愿望——约会体验设计的最好素材，见 `references/dating.md`；只记 ta 自己流露过的，别脑补）。

用法：底色定语气底盘（偏理性少灌情绪长文、偏情绪先接感受）；求稳的人别用猛拉扯；底色比阶段稳，但出现矛盾证据时按 `references/memory.md` 的置信度规则更新，别一条消息推翻整层。

## 八、话术素材与自带话术校准

素材只是骨架，输出前一律过 `references/voice.md` 的人话门禁 + 本文件的油腻度上限 + 气质匹配 + 像用户本人四道。标注：`[气质 油腻度]`。

**开场 / 破冰**
- `[通用 0.5]` 顺朋友圈具体细节起话：`你那张图的店在哪 看着不错`
- `[痞 1]` `我猜你朋友圈和本人反差挺大`
- `[乖 0.5]` `刚刷到个东西想到你上次说的 来问问`

**接住情绪（ta 说累 / emo）**
- `[通用 1]` `累成这样` + `今晚必须吃点好的回血`
- `[痞 1]` `这么累还回我 感动住了`
- `[乖 1]` `那先瘫着 想吐槽随时说`

**轻撩 / 留余味（注意阶段上限）**
- `[通用 1]` `跟你聊天有点上头 得收着点`
- `[痞 1.5]` `你这人有点难搞 还好我脾气好`
- `[乖 1]` `怎么跟你一聊就聊到这个点了`
- 收尾留钩：`先去忙了 后半段明天讲`

**邀约（落到具体，不悬在「改天」）**
- `[通用 1]` `周六下午有个展 顺路 要不要一起`
- `[通用 1]` `这周找天喝咖啡 我知道一家还行的`
- `[乖 1]` `那家你想去的店我查了下 周末人少 去不去`
- 被挡不加码：`行 那等你有空 不急`

**拉扯 / 回温**
- 久不回后回来：`[痞 1]` `稀客啊`
- 画饼不落地：`[通用 1]` `行啊 那定哪天`
- 展示自己：`[通用 0.5]` `刚爬完山 腿已经不是我的了` + `但山顶是真的绝`

**约会后跟进**
- `[通用 1]` `今天挺好玩的` + `下次去那家我说的店`
- `[乖 1]` `到家了说一声`

**自带话术校准**：用户从短视频/评论区看来想用的句子，丢进来按四个维度判断——油腻度超没超上限、阶段合不合、对方吃痞还是吃乖要不要换味道、像不像用户本人会说的话。输出：能用 / 改一下能用 / 不建议，附改写版和一句理由。学博主只学「为什么不油、节奏怎么留白」，原文一律现改现用，整段照搬对方一眼能看出来。

## 九、评估用户想说的话（/ask）

用户说「我想发 XXX，行不行」时，不生成新方案，先评估 ta 的原话。

**输出格式**：

```markdown
## 军师评估
- 你想说："{原话}"　当前阶段：{阶段名}
- 结论：**{可以说 / 需要调整 / 不建议}**
- 阶段适配 {N}/10 · 油腻度 {N}/5（上限 {M}/5）· 需求感 {低/中/高} · 撩感 {会撩/有点油/压迫感强}
{有翻车点就一句话点出来}

### 建议说法
**方案 1 · {调整方向}**　{优化后的话}　为什么更好：{一句}
**方案 2 · {调整方向}**　{优化后的话}　为什么更好：{一句}
{如适合}**暂时别说**：等 {时机} 再开口，{原因}

### 时机
- 适合：{最佳时间/场景}　别选：{不适合的}
```

**「不建议」触发条件**（任一满足）：油腻度超上限 1.5 以上；早期严重暴露需求感；逼对方表态或无法回答；查岗、翻旧账、情绪逼答案；已触发反舔狗硬阈值（见 `references/reading.md`）；想用文艺长文弥补对方的行动缺席。

**「需要调整」**：意思对但需求感太明显、油腻度轻微超标、时机不对、撩点太满不像本人。

**改写优先级**：保留真实意图 → 降油腻和压迫 → 指责改观察、逼问改确认、索取改邀请 → 想撩就留一点余味 → 像聊天不像心理咨询。经典改写：「你是不是不想理我了」→「你消失得有点彻底 我还以为把天聊死了」。

**特殊场景**：

- **想提前表白**（阶段 1-2）：直说还不是时机——对方没准备好只能本能拒绝，主动交底后从此弱势。给传递好感但不摊牌的替代说法，等对方也有明显主动信号。
- **想问「你喜欢我吗」**：不建议。把不确定性变成必答题，对方没准备好只能说不。给创造对方主动表达的场景。
- **反舔狗模式开启且对方连续低反馈**：直接拦——「这不是你缺一句更漂亮的话，是对方没给动作。今天别主动了」。证据够硬就别安慰。
- **想道歉/挽回**（阶段 6-7）：先确认 ta 知道对方真正在意什么。有效道歉三件事：承认具体的事（不是泛泛"我错了"）、说明理解对方为什么受伤、说明会怎么做（不是"我会改"）。
